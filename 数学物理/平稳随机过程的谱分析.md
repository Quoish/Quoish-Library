#随机过程 
# 随机过程的功率
随机过程的功率也有统计平均与时间平均之分。
统计平均就是均方值，如果不是平稳过程，那就会与时间有关$$P(t)=E[X^2(t)]=R_{X}(0,0)$$时间平均要对一个特定样本$x_{i}(t)$在整个时间轴上做平均，要求是一个常数$$P_{avg}=\langle x_{i}^2(t) \rangle = \lim_{ T \to \infty } \frac{1}{2T}\int_{-T}^{+T}x_{i,2T}^2(t)\mathrm{d}t$$这里$x_{i,2T}(t)$指的是截取过程中一段长度为$2T$的部分。
若随机过程是遍历过程，则$$\langle X^2(t) \rangle \overset{P}{\longrightarrow} E[X^2(t)]=P$$定义一个样本的功率谱密度函数$G_{i}(\omega)$$$P_{i}
=\lim_{T\to\infty}\frac{1}{T}\int^{\frac{T}{2}}_{-\frac{T}{2}}x_{i}^2(t)\text{d}t
=\frac{1}{2\pi}\int^{+\infty}_{-\infty}\lim_{T\to\infty}\frac{|X_{i}(j\omega)|^2}{T}\text{d}\omega
=\frac{1}{2\pi}\int^{+\infty}_{-\infty}G_{i}(\omega)\text{d}\omega$$因为是遍历过程，可以取平均$$G_{X}(\omega)=E\left[ \frac{1}{2\pi} \int^{+\infty}_{-\infty}\lim_{T\to\infty}\frac{|X_{i}(j\omega)|^2}{T}\text{d}\omega\right]$$这就是随机过程的功率谱密度函数。
# 功率谱密度的性质
- [[相关函数#Wiener-Khinchin定理]]：$$G_{X}(\omega)=\mathscr{F}[R_{X}(\tau)]$$
- 若$X(t)$是实平稳随机过程，则$R_{X}(\tau)$是实偶函数，自然$G_{X}(\omega)$也是实偶函数，有$$G_{X}(-\omega)=G_{X}(\omega)\geq0$$

$G_{X}(\omega)$是定义在$-\infty<\omega<+\infty$上的，但是实际上负频率并不存在，实际功率只会由正频率分量所产生。这里的负频率的概念，认为负频率分量与正频率分量同样地产生功率，是为了便于数学分析，因此$G_{X}(\omega)$称为数学功率谱密度或*双边功率谱密度*。
实际得到的功率谱密度都只会分布在的正频率域上，这种功率谱密度称为物理功率谱密度或*单边功率谱密度*，以$F_{X}(\omega)$记之，有关系式$$F_{X}(\omega)=2G_{X}(\omega),\quad \omega>0$$
# 联合平稳过程的互谱密度
设有两个联合平稳的随机过程$X(t)$和$Y(t)$，其任一样本的截段函数分别为$x_{Ti}(t)$和$y_{Ti}(t)$，它们的频谱密度随样本编号$i$的不同而分别为随机变量$X_{Ti}(\omega)$和$Y_{Ti}(\omega)$，定义:$$G_{XY}(\omega)=E\left[ \lim_{ T \to \infty }  \frac{X_{T}^*(\omega)Y_{T}(\omega)}{2T} \right],\quad G_{YX}(\omega)=E\left[ \lim_{ T \to \infty }  \frac{Y_{T}^*(\omega)X_{T}(\omega)}{2T} \right]$$称$G_{XY}(\omega)$或$G_{YX}(\omega)$为随机过程$X(t)$与$Y(t)$的互谱密度。
