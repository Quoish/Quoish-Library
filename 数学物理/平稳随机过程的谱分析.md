# 随机过程的功率
随机过程的功率也有统计平均与时间平均之分。
统计平均就是均方值，如果不是平稳过程，那就会与时间有关$$P(t)=E[X^2(t)]=R_{X}(0,0)$$时间平均要对一个特定样本$x_{i}(t)$在整个时间轴上做平均，要求是一个常数$$P_{avg}=\langle x_{i}^2(t) \rangle = \lim_{ T \to \infty } \frac{1}{2T}\int_{-T}^{+T}x_{i}^2(t)\mathrm{d}t$$
若随机过程是遍历过程，则$$\langle X^2(t) \rangle \overset{P}{\longrightarrow} E[X^2(t)]=P$$定义一个样本的功率谱密度函数$G_{i}(\omega)$$$P_{i}
=\lim_{T\to\infty}\frac{1}{T}\int^{\frac{T}{2}}_{-\frac{T}{2}}x_{i}^2(t)\text{d}t
=\frac{1}{2\pi}\int^{+\infty}_{-\infty}\lim_{T\to\infty}\frac{|X_{i}(j\omega)|^2}{T}\text{d}\omega
=\frac{1}{2\pi}\int^{+\infty}_{-\infty}G_{i}(\omega)\text{d}\omega$$因为是遍历过程，可以取平均$$G(\omega)=E\left[ \frac{1}{2\pi} \int^{+\infty}_{-\infty}\lim_{T\to\infty}\frac{|X_{i}(j\omega)|^2}{T}\text{d}\omega\right]$$这就是随机过程的功率谱密度函数。
