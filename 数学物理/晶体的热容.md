#固体物理 #统计物理 
定容热容定义为$$C_{V}=\left( \frac{ \partial  U }{ \partial  T}  \right)_{V}$$
晶体中，内能$U$包括晶格振动的能量$U_{1}$和电子运动的能量$U_{e}$。这两种运动的能量对定容热容的贡献分别用$C_{V1}$和$C_{Ve}$表示。
但除了低温情况下，通常$C_{V1}\gg C_{Ve}$。所以本文讨论的$C_{V}$指的直接就是$C_{V 1}$。
# 固体热容的经验定律
## 高温的Dulong-Petit定律
对于确定的材料，高温下的热容为常数，有摩尔热容$c_{V}=3R$，其中R是理想气体常数。
这个可以通过经典理论的[[能量均分定理]]得到。
设单位质量的晶体中有$sN$个原子，则其自由度数为$3sN$。由以前的分析可知，晶体中的格波可归结为$3sN$个相互独立的简谐振子。根据能量均分定理，每个简谐振子的平均e能量为$k_{B}T$，因而总晶格振动能为$$U=3sNk_{B}T$$
相应的热容为$$C_{V}=3sNk_{B}$$
摩尔热容为$$c_{V}=3N_{A}k_{B}=3R$$
其中$N_{A}$为Avogadro常数。
## 低温的Debye定律
低温下的固体热容与$T^3$成正比。
这个无法被经典理论所解释，直到量子理论出现后，才能解释其原理。
## 固体热容的量子理论
根据量子力学中[[一维线性谐振子]]的理论，线性谐振子的能量不再是$k_{B}T$，而是$$E(\omega,T)=\left( \frac{1}{2}+\bar{n} \right)\hbar \omega=\left( \frac{1}{2}+\frac{1}{e^{\hbar \omega/k_{B}T}-1} \right)\hbar \omega$$
晶格振动能于是变成$$U=\sum_{i=1}^{3sN}E_{i}=\sum_{i=1}^{3sN}\left( \frac{1}{2}+\frac{1}{e^{\hbar \omega_{i}/k_{B}T}-1} \right)\hbar \omega_{i}$$
使用格波态密度函数$g(\omega)$，会有以下关系
$$
\begin{equation}
\left\{
\begin{aligned}
&U=\int_{0}^{\omega_{m}}g(\omega)E(\omega,T)\mathrm{d}\omega\\
&3sN=\int_{0}^{\omega_{m}}g(\omega)\mathrm{d}\omega\\
\end{aligned}
\right.
\end{equation}
$$
于是，
$$C_{V}=\frac{ \partial   }{ \partial  T}\int_{0}^{\omega_{m}}g(\omega)E(\omega,T)\mathrm{d}\omega=\int_{0}^{\omega_{m}}k_{B}\left( \frac{\hbar \omega}{k_{B}T} \right)^2 \frac{e^{ \hbar \omega /k_{B}T}}{(e^{ \hbar \omega/k_{B}T }-1)^2}g(\omega)\mathrm{d}\omega $$
可见，热容问题的核心就是求解格波态密度函数$$g(\omega)=\sum_{i=1}^{3s} \frac{V}{(2\pi)^3}  \frac{4\pi k^2(\omega)}{\omega_{i}'(k)}$$
对于具体的晶体，求出格波态密度函数$g(\omega)$十分困难，求解$C_{V}$的积分也很不容易。
所以人们常常使用简化模型来讨论晶体热容问题。
# Einstein模型
>假设：所有原子都以相同的频率$\omega$振动，则晶体中的格波频率都相同。

依此假设，$sN$个原子组成的内能为$$U=3sNE(\omega,t)=3sN\left( \frac{1}{2}+\frac{1}{e^{\hbar \omega/k_{B}T}-1} \right)\hbar \omega$$
则热容为$$C_{V}=\left( \frac{ \partial  U }{ \partial  T}  \right)_{V}=3sNk_{B}\left( \frac{\hbar \omega}{k_{B}T} \right)^2 \frac{e^{ \hbar \omega/k_{B}T }}{(e^{ \hbar \omega/k_{B}T}-1)^2}$$
引入Einstein特征温度$\hbar \omega=k_{B}\Theta_{E}$。$\Theta_{E}$跟$\omega$是直接关联的，因此跟具体的固体有关，一般在$100\mathrm{K}$到$300\mathrm{K}$之间。
将$C_{V}$的表达式改写为$\Theta_{E}$和$T$的函数$$C_{V}=3sNk_{B}\left( \frac{\Theta_{E}}{T} \right)^2 \frac{e^{ \Theta_{E}/T }}{(e^{ \Theta_{E}/T }-1)^2}$$
这就是Einstein模型近似下的热容公式。
**高温的情况**：
考虑高温时$T\gg \Theta_{E}$，$e^{ \Theta_{E}/T }-1\approx \frac{\Theta_{E}}{T}$。
Einstein热容公式将变为$$C_{V}=3sNk_{B}$$这就是Dulong-Petit定律描述的，能量均分定理的结论。
**低温的情况**：
考虑低温时$T \ll \Theta_{E}$，$e^{ \Theta_{E}/T }-1\approx e^{ \Theta_{E}/T }$。
Einstein热容公式将变为$$C_{V}=3sNk_{B}\left( \frac{\Theta_{E}}{T} \right)^2e^{ -\Theta_{E}/T }$$
这表明$$\lim_{ T \to 0 } C_{V}=0$$
物理上解释，这是因为温度趋于绝对零度时，谐振子的能级间距将远大于$kT$。谐振子必须要取得至少$\hbar \omega$的能量才能向上一能级跃迁，但在如此的低温下，取得这样的能量的概率是很小的，这个时候所有谐振子几乎全部凝结在基态。因此即使上升温度，但达不到$\hbar \omega$，谐振子也不会吸收能量。
# Debye模型
>假设：晶体是各向同性的连续弹性媒质。这意味着不把晶体看成离散的点阵，而是连续的介质，波在其中的传播就是简单的行波，色散关系是线性的，且忽略掉了光学支格波。

设晶体是N个初基原胞组成的三维单式格子（$s=1$），此时晶体中只有3支声学支格波，并设它们相速度相同。因而三支格波的色散关系都是线性的：$$\omega= v_{p}k$$
则等频率面为球面$$|\nabla_{k}\omega(\vec{k})|=\frac{ \mathrm{d}  \omega }{ \mathrm{d}  k} =v_{p}$$
所以格波态密度函数为$$g(\omega)=\sum_{i=1}^{3s} \frac{V}{(2\pi)^3}  \frac{4\pi k^2(\omega)}{\omega_{i}'(k)}=3\cdot \frac{V}{(2\pi)^3}\cdot \frac{4\pi k^2}{v_{p}}=\frac{3V}{2\pi^2v_{p}^3}\omega^2$$
然后热容为$$C_{V}=\frac{ \partial   }{ \partial  T}\int_{0}^{\omega_{m}}g(\omega)E(\omega,T)\mathrm{d}\omega= \frac{3V}{2\pi^2 v_{p}^3}\int_{0}^{\omega_{m}}k_{B} \left( \frac{\hbar \omega}{k_{B}T} \right)^2 \frac{e^{ \hbar \omega/k_{B}T }}{(e^{ \hbar \omega/k_{B}T }-1)^2}\omega^2\mathrm{d}\omega$$
其中截止频率$\omega_{m}$又称为Debye频率，记为$\omega_{D}$，由格波总数等于$3N$来确定：$$\int_{0}^{\omega_{D}}g(\omega)\mathrm{d}\omega=3N \implies \omega_{D}^3= \frac{6\pi^2Nv_{p}^3}{V}$$
引入Debye温度$h\omega_{D}=k_{B}\Theta_{D}$，变量代换：
$$\begin{equation}
\left\{
\begin{aligned}
&x=\frac{\hbar \omega}{k_{B}T}\\
&\mathrm{d}\omega=\frac{k_{B}T}{\hbar}\mathrm{d}x
\end{aligned}
\right.
\end{equation}$$
则热容计算公式可以改写为$$C_{V}=9Nk_{B} \frac{T^3}{\Theta_{D}^3}\int_{0}^{\Theta_{D}/T} \frac{x^4e^{ x }}{(e^{ x }-1)^2}\mathrm{d}x$$
这就是Debye模型近似下的热容公式。
**高温的情况**：
考虑高温时$T\gg \Theta_{D}$，$e^{ \Theta_{D}/T }-1\approx \frac{\Theta_{D}}{T}$。
则Debye热容公式中的积分变为$$\int_{0}^{\Theta_{D}/T} \frac{x^4 e^{ x }}{(e^{ x }-1)^2}\mathrm{d}x\approx \int_{0}^{\Theta_{D}/T} \frac{x^4(1+x)}{x^2}\mathrm{d}x\approx \int_{0}^{\Theta_{D}/T}x^2 \mathrm{d}x$$
所以Debye热容公式变为$$C_{V}=9Nk_{B} \left( \frac{T}{\Theta_{D}} \right)^3 \cdot \frac{1}{3}\left( \frac{\Theta_{D}}{T} \right)^3=3Nk_{B}$$
这就是Dulong-Petit定律描述的，能量均分定理的结论。
**低温的情况**：
考虑低温时$T \ll \Theta_{D}$，$\Theta_{D}/T \to \infty$。
则Debye热容公式中的积分上限将可以取为无穷大$$\int_{0}^{\Theta_{D}/T} \frac{x^4 e^{ x }}{(e^{ x }-1)^2}\mathrm{d}x\approx \int_{0}^{+\infty } \frac{x^4 e^{ x }}{(e^{ x }-1)^2}\mathrm{d}x=\frac{4\pi^2}{15}$$
所以Debye热容公式变为$$C_{V}=9Nk_{B}\left( \frac{T}{\Theta_{D}} \right)^3\cdot \frac{4\pi^4}{15}=\frac{12\pi^4}{5}Nk_{B}\left( \frac{T}{\Theta_{D}} \right)^3$$
这与Debye实验定律的结果相符合。
# 两种模型的比较
![[两种热容模型的比较.png]]
实验表明，在低温情况下，Debye模型要比Einstein模型拟合得更好。
这是因为Einstein模型做了过分简化的假设。在[[晶体声子论#平均声子数]]处，我们认为只有$\hbar\omega>k_{B}T$时，声子才算被激发，才算参与了热力学过程，在这里也就是对热容有了贡献。Einstein模型中假设所有原子都以相同的频率独立振动。从绝对零度起开始上升温度，在Einstein模型的视角下，所有声子就像仿佛突然间从全部未被激发到全部被激发了一样。这和实际情况是不符的，肯定是先有部分声子被激发，然后再慢慢地全部被激发。
而Debye考虑了格波的频率分布，并把晶体当做连续弹性介质来处理。温度越低，声子的能量就越低，格波的波长就越长，在长波看来，晶体离散的间隔就不那么重要了，就越接近连续介质的假设。所以温度越低，Debye模型越接近实际情况。
实际上，$C_{V}\propto T^3$的规律只适用于$T< (\frac{1}{30}\sim\frac{1}{12})\Theta_{D}$的情况，也就是绝对温度几度以下的极低温情况。
这两种模型都是正确的，但是相对而言Einstein模型更简单，因此在高温情况下常采用Einstein模型，低温情况下常采用Debye模型。
对于复式格子的情况，我们可以将两种模型结合起来，Debye模型用于声学支格波，Einstein模型用于光学支格波，原因是很多晶体的光学支格波频带都很窄，各格波振动频率相差不大，可以把各光学支格波对热容的贡献近似视为$3N(s-1)$个独立同频率的谐振子的贡献，一般不会导致太大误差。
# Debye温度的性质
### Debye温度高于Einstein温度$\Theta_{D}>\Theta_{E}$
从上面的讨论可以知道，Einstein频率$\omega_{E}$是格波态密度函数的最可几频率，而Debye频率$\omega_{D}$是截止频率，所以$\omega_{D}>\omega_{E}$，对应$\Theta_{D}>\Theta_{E}$。
### Debye温度是热容问题中经典和量子理论的分界线
线性谐振子系统的能级间隔$\Delta \varepsilon=\hbar \omega$，当$\hbar \omega>k_{B}\Theta_{D}$时，我们可以用经典理论来处理热容问题。
### Debye温度与其他物理性质的关联
一般晶体越硬，密度越低，则弹性波的速度越大，Debye频率也越高，Debye温度也越高。如金刚石、硼、铍的$\Theta_{D}$高达$1000\mathrm{K}$以上。
金和铅有较低的Debye温度，所以可以说，室温对于它们来说就相当于“高温”了，所以金和铅在室温下有很好的延展性。
### Debye温度的经验公式
Lindeman总结出了计算Debye温度的如下经验公式$$\Theta_{D}\approx C\left[ \frac{T_{M}}{A\times V^{2/3}} \right]^{1/2}$$
其中$C$是常数，$T_{M}$是材料熔点（单位为开尔文），$A$为原子量，$V$为原子体积。
### Debye理论的局限
若Debye理论是精确成立的，则对于不同的温度，Debye温度应该不会发生变化。
但实验发现不同温度得到的$\Theta_{D}$是不同的，其变化量可达到10%。
这时为了获得更精确的公式可能要采用完整的热容公式。
