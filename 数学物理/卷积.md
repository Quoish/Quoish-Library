# 卷积积分
连续信号的卷积积分定义为$$(f_1*f_2)(t)=f_1(t)*f_2(t)=\int_{-\infty}^{+\infty}f_1(\tau)f_2(t-\tau)\mathrm{d}\tau$$
在计算时可以采用图解法：
1. 换元：将$f_1(t)$和$f_2(t)$换元成$f_1(\tau)$和$f_2(\tau)$。
2. 反转：将$f_2(\tau)$依纵轴镜像对称，得到$f_2(-\tau)$。
3. 平移：将$f_2(\tau)$向右平移$t$个单位，得到$f_2(t-\tau)$。
4. 求积：分区间求相乘的积分。

例题：求图中所示函数$f_{1}(t)$和$f_{2}(t)$的卷积积分。![[卷积积分例题.png]]换元、反转、平移后的图像如图所示。
![[换元反转平移波形.png]]
不断平移，分段卷积可以得到$$f(t)=f_{1}(t)*f_{2}(t)=\begin{equation}
\left\{
\begin{aligned}
&0,&t<-2\\
&\int_{-2}^{t}2\times \frac{3}{4}\mathrm{d}\tau=\frac{3}{2}(t+2),&-2<t<0\\
&\int_{t-2}^{t}2\times \frac{3}{4}\mathrm{d}\tau=3,&0<t<2\\
&\int_{t-2}^{2}2\times \frac{3}{4}\mathrm{d}\tau=\frac{3}{2}(4-t),&2<t<4\\
&0,&t>4
\end{aligned}
\right.
\end{equation}$$![[卷积计算结果.png]]
# 卷积和
离散信号的卷积和定义为$$f_1(k)*f_2(k)=\sum_{i=-\infty}^{+\infty}f_1(i)f_2(k-i)$$
计算卷积和也可以用图解法，不过最方便的方法是利用乘法竖式。
例题：求序列$f_{1}(k),f_{2}(k)$的卷积和，其中$f_{1}(k)=\{ \dots,0,\underline{1},2,3,0,\dots \}$，$f_{2}(k)=\{ \dots,0,\underline{1},1,1,1,0,\dots \}$。

# 圆卷积
有限序列的$N$位圆卷积定义为$$f_{1}(k) \circledast f_{2}(k)=\left[ \sum_{m=0}^{N-1} f_{1}(m) f_{2}((k-m))_{N} \right]R_{N}(k)=\left[ \sum_{m=0}^{N-1} f_{2}(m) f_{1}((k-m))_{N} \right]R_{N}(k)$$如果两个有限序列$f_{1}(k),f_{2}(k)$的长度分别为$N_{1},N_{2}$，且取圆卷积区间长度$N\geq \max\{ N_{1},N_{2} \}$，则圆卷积是通过周期延拓，使两个有限序列都变成周期为$N$的周期序列，然后在一个周期内做卷积和，再取主值区间，得到的序列长度仍为$N$。
