#热力学 #物理化学 
# 热力学第二定律
热力学第二定律的主要内容是，凡牵扯到热现象的过程都是不可逆的。
热力学有个基本的现象就是趋向平衡态，这是一个显著的不可逆过程。当两个物体接触时，热量总是从高温物体流向低温物体，从来不反向流，这说明热传导是不可逆的。Joule测量热功当量的实验中，摩擦生热也是不可逆的。
大量经验事实指出，不可逆过程产生的效果，无论经过多么曲折复杂的方法，都不可能完全恢复原状而不产生其他变化，这是热力学第二定律的基础，其正确性由无数热力学第二定律的推论都与实验相符合而得到保证。
热力学第二定律有两种表述方式，分别来自William Thomson（第一代Kelvins男爵）、Rudolf Clausius。Kelvins表述相当于热传导不可逆，Clausius表述相当于摩擦生热不可逆：
1. Kelvins表述：不可能把热从低温物体传到高温物体而不引起其他变化。
2. Clausius表述：不可能从单一热源取热使之完全变为有用的功而不引起其他变化。

此外，还有两种等价的说法：

3. Plank表述：不可能造一个机器，在循环动作中把一重物升高而同时使一热库冷却。
4. Caratheodory表述：在一个物体系的任一给定的平衡态的附近总有这样的态存在，从给定的态出发不可能经绝热过程达到。
# 循环过程
热力学研究的重要课题就是怎样能提高热机的效率。
*热机*指的就是通过某种工质（如气体）不断把吸收的能量转变为机械功的装置，如蒸汽机，内燃机等。
如果物质系统的状态经历了一系列变化之后，又回到了原来的状态，就称经历了一个*循环过程*。
热机就是工质在经历循环过程中，物质通过某种方式提高自身内能，然后部分内能通过对外做功转化为机械能，另一部分通过放热传给外界的机器。
循环过程可以用P-V图上的一个闭合曲线来表示，顺时针的称为*正循环*，逆时针的称为*逆循环*。
- 正循环中，系统对外界做功$A$，其值为闭曲线所围的面积。系统从高温热源吸收的热量$Q_{AH}$和释放给低温热源的热量$Q_{RC}$应满足$$A=Q_{AH}-Q_{RC}$$热机吸收的能量用于做功的比例越高，说明能量利用效率越高。定义*热效率*$$\eta=\frac{A}{Q_{AH}}=\frac{Q_{AH}-Q_{RC}}{Q_{AH}}=1-\frac{Q_{RC}}{Q_{AH}}$$
- 逆循环中，外界对系统做功$A'$，其值为闭曲线所围的面积。系统从低温热源吸收的热量$Q_{AC}$和释放给高温热源的热量$Q_{RH}$应满足$$A'=Q_{RH}-Q_{AC}$$外界对制冷机做的功相比吸收的热量比例越高，说明能量利用效率越高。定义*制冷效率*$$\varepsilon=\frac{Q_{AC}}{A'}=\frac{Q_{AC}}{Q_{RH}-Q_{AC}}$$
- 两种效率的关系是$$\frac{1}{\eta}-1=\frac{Q_{AH}}{Q_{AH}-Q_{RC}}-1=\frac{Q_{RC}}{Q_{AH}-Q_{RC}}=\frac{-Q_{AC}}{-Q_{RH}+Q_{AC}}=\frac{Q_{AC}}{Q_{RH}-Q_{AC}}=\varepsilon$$

## Carnot循环
热机的Carnot循环经历四个过程：
1. 绝热膨胀（$A\to B$）：由高温$T_{1}$到低温$T_{2}$；
2. 等温压缩（$B\to C$）：在恒温容器中放热$Q_{2}$；
3. 绝热压缩（$C\to D$）：由低温$T_{2}$到高温$T_{1}$；
4. 等温膨胀（$D\to A$）：在恒温容器中吸热$Q_{1}$。

### Carnot定理
Carnot在热力学第二定律发现之前，就曾基于错误的热质说提出过Carnot定理，不过在热力学第二定律提出后，这些定理被证明是正确的。
Carnot定理的内容包括：
1. 一切在两个给定温度$T_{C},T_{H}$（$T_{C}<T_{H}$）热源之间工作的热机，其效率都不会超过可逆热机；
2. 一切可逆热机的效率都相同；
3. 理想气体可逆卡诺热机效率为$$\eta=1 - \frac{T_{C}}{T_{H}}$$其他一切可逆热机的效率也是这个。

---
证明：
设有两个热机$R,X$，它们在同样的高温热源和低温热源之间运行。它们的效率分别是$\eta_{R}=A/Q_{AH},\eta_{X}=A'/Q_{AH}'$。
假如$R$是可逆机，$X$是比它热效率更高的机器，即$\eta_{R}<\eta_{X}$。
我们首先运行$X$，从高温热源吸热$Q_{AH}$，向低温热源放热$Q_{RC}$，向外做功$A=Q_{AH}-Q_{RC}$。
然后我们反向运行$R$，消耗功$A'$从低温热源吸热$Q_{AC}'$，向高温热源放热$Q_{RH}'$，消耗功$A'=Q_{RH}'-Q_{AC}'$。
假设$A=A'$，让$X$做的功供给$R$。
对于高温热源，吸收的热量为$\Delta Q_{H}=Q_{RH}'+Q_{AH}=-Q_{AH}'+Q_{AH}$。
对于低温热源，吸收的热量为$\Delta Q_{C}=Q_{RC}'+Q_{AC}=-Q_{AC}'+Q_{AC}$。
因为[[热力学第一定律]]或能量守恒，两个热机的对外做功为零，所以总的热量没有变化，则$$\Delta Q_{H}+\Delta Q_{C}=-Q_{AH}'+Q_{AH}-Q_{AC}'+Q_{AC}=0\implies Q_{AH}'-Q_{AH}=Q_{RC}'-Q_{RC}$$又因为$\eta_{R}<\eta_{X}$，且$$A=A' \implies \eta_{X}Q_{AH}'=\eta_{R}Q_{AH} \implies Q_{AH}'=\frac{\eta_{R}}{\eta_{X}}Q_{AH}$$所以$Q_{AH}'-Q_{AH}=Q_{RC}'-Q_{RC}<0$，也就是$$\Delta Q=Q_{RH}'-Q_{RH}=Q_{AC}'-Q_{AC}>0$$所以经过这两步，从不带撇到带撇的变化，相当于从低温热源吸收了$\Delta Q$的热量，向高温热源释放了$\Delta Q$的热量，而系统总功为零，没有产生任何别的变化。这违反了热力学第二定律的Clausius表述。所以只能$\eta_{R}\geq \eta_{X}$。
就此我们证明了可逆热机的效率是最大的。最大值只有一个，自然，所有可逆热机的效率是一样的。
使用[[状态方程#理想气体状态方程]]和[[热力学第一定律]]，经过一系列计算即可得到，理想气体可逆卡诺热机的效率为$$\eta=1 - \frac{T_{C}}{T_{H}}$$自然，对于一切可逆热机，效率都是如此。
## Otto循环
![[奥托循环.png]]
Otto循环在内燃机中广泛运用。
热机的Otto循环经历四个过程：
1. 绝热压缩（$\mathrm{I}\to \mathrm{II}$）：气缸被活塞迅速压缩；
2. 等容升温（$\mathrm{II}\to \mathrm{III}$）：火花塞点火，气缸内气体迅速升温，来不及膨胀；
3. 绝热膨胀（$\mathrm{III}\to \mathrm{IV}$）：气体膨胀带动活塞迅速运动；
4. 等容降温（$\mathrm{IV}\to \mathrm{I}$）：气体放热回到最初状态。

其效率可以计算出来，仅与气体压缩比$r=V_{1}/V_{2}$以及比热容比$\gamma$有关$$\eta=1-\frac{1}{r^{\gamma-1}}$$
## Stirling循环
![[斯特林循环.png]]
热机的Stirling循环经历四个过程：
1. 等温膨胀（$1\to 2$）
2. 等容降温（$2\to 3$）
3. 等温压缩（$3\to 4$）
4. 等容升温（$4\to 1$）

在理想状态下，Stirling循环的效率和Carnot循环完全相同$$\eta=1 - \frac{T_{C}}{T_{H}}$$
# Clausius不等式
根据Carnot定理，任意Carnot循环的效率必定小于可逆Carnot循环的效率$$\eta=1-\frac{Q_{RC}}{Q_{AH}}\leq1 - \frac{T_{C}}{T_{H}}$$从这里可以得到$$\frac{Q_{AH}}{T_{H}}-\frac{Q_{RC}}{T_{C}}\leq 0$$假如都写成吸热的形式，就变成$$\frac{Q_{AH}}{T_{H}}+\frac{Q_{AC}}{T_{C}}\leq 0$$现在，我们考虑对于任意一个循环过程。![[克劳修斯不等式证明.png]]任意一个循环过程可以看作许许多多相邻的Carnot循环组合而成的。这些Carnot循环中重叠的路径方向相反，所以可以互相抵消，最终就只剩下边缘那些锯齿状的路径。我们可以取大量微小的Carnot循环，使得边缘变得光滑，最终就能拟合任意一个循环过程的路径。
这些微小的Carnot循环总共在跟$n$个温度分别为$T_{i}$的不同热源接触，吸收的热量分别为$Q_{i}$，将上面那样的方程依次叠加可以得到$$\sum_{i} \frac{Q_{i}}{T_{i}}\leq 0$$极限情况下，设$\mathrm{d}Q$是很小的热量，则这些加和可以表示为一个路径积分$$\boxed{\oint \frac{\mathrm{d}Q}{T}\leq 0}$$这就是*Clausius不等式*，其中等号仅在Carnot循环效率最大，也就是过程可逆的时候取到。
# 熵
由Clausius不等式，我们可以发现，当过程为可逆的时候$$\oint \frac{\mathrm{d}Q}{T}=0$$这说明，这里有一个量$\mathrm{d}Q/T$它是路径无关的，只和初态和末态有关。
Clausius将这个路径无关的状态量定义为*熵*：$$\mathrm{d}S=\frac{\mathrm{d}Q}{T}$$
引入了熵以后，我们就可以得到[[热力学第一定律]]的完全形态：$$\mathrm{d}U=T\mathrm{d}S-p\mathrm{d}V$$不过至此，和内能一样，因为势函数的性质，我们只能将熵定义到相差一个任意常数。
假设$P_{0}$到$P$的过程为可逆过程，则$$\int_{P_{0}}^{P} \frac{\mathrm{d}Q}{T}=S-S_{0}$$但在[[Maxwell-Boltzmann分布#熵]]中，我们会揭示Boltzmann关系，这时我们可以定义熵的绝对数值。
## 熵增原理
我们分三种情况来讨论熵增原理的证明过程。
### 初末态都是平衡态
设体系由平衡态$P_{0}$发展到平衡态$P$，因为初态和末态都是平衡态，我们总可以找到一个连续的、准静态的、可逆的过程，从末态慢慢恢复到平衡态。
根据Clausius不等式，有$$\int_{P_{0}}^{P} \frac{\mathrm{d}Q}{T}+\int_{P}^{P_{0}} \frac{\mathrm{d}Q}{T}\leq 0$$代入熵的定义$$\int_{P_{0}}^{P}\frac{\mathrm{d}Q}{T}\leq -\int_{P}^{P_{0}}\frac{\mathrm{d}Q}{T}=S-S_{0}$$如果$P_{0}$到$P$是绝热过程，则$\int_{P_{0}}^{P}\frac{\mathrm{d}Q}{T}=0$，故$$S-S_{0}\geq 0$$所以得出：*绝热过程的熵永不减小*。
### 初末态不是平衡状态
即使初末态不是平衡状态，我们也可以把体系分成许多小部分，每一个小部分可以近似认为处在平衡态，每个小部分也有熵函数。体系总的熵就是各个小部分的熵加起来，所以熵增原理依然成立
### 孤立系
孤立系不受外界影响，过程当然是绝热的，所以熵增原理成立。
熵增原理的一个常用的说法就是：*孤立系的熵永不减小*。
## 熵作为过程方向的判据
熵增原理可以用来判断过程是否可逆以及进行的方向。
- 绝热过程：
	- 熵相等：可逆
	- 熵增加：可以进行
	- 熵减小：不可以进行
- 非绝热过程：
	- 把系统和外界看成一个整体，然后用一样的方法判断。
