#量子力学 
# 一般情况
假设Hamilton量不含时，则能量本征方程为$$\hat{H}\ket{\psi}=E\ket{\psi}$$可以将Hamilton算符分成两部分$$\hat{H}=\hat{H}_{0}+\hat{H}'$$其中$\hat{H}^{(0)}$可解体系的Hamilton算符，已经求解$\hat{H}^{(0)}\ket{\psi^{(0)}}=E^{(0)}\ket{\psi^{(0)}}$，而$\hat{H}'$则是作为微扰的Hamilton量。
可以将波函数与能量视作许多阶近似的展开：
$$\begin{equation}
\left\{
\begin{aligned}
&\ket{\psi} =\ket{\psi^{(0)}}+\ket{\psi^{(1)}}+\ket{\psi^{(2)}}+\dots\\
&E=E^{(0)}+E^{(1)}+E^{(2)}+\dots
\end{aligned}
\right.
\end{equation}$$
并且约定，波函数的各级彼此正交$\langle \psi^{(m)}|\psi^{(n)} \rangle=\delta_{mn}$。
把展开式代入本征方程，比较等式两边的同阶项。所谓同阶项说的是，“微扰阶数”的同阶，确定微扰阶数的规则如下：
- $\hat{H}^{(0)}$、$\ket{\psi^{(0)}}$、$E^{(0)}$被视为零阶量；
- $\hat{H}'$被视为一阶量；
- $\ket{\psi^{(n)}}$和$E^{(n)}$被视为$n$阶量；
- $m$阶量与$n$阶量相乘，得到的是$m+n$阶量。

由此从$$(\hat{H}_0 + \hat{H}') (\ket{\psi^{(0)}} + \ket{\psi^{(1)}} + \ket{\psi^{(2)}} + \dots) = (E^{(0)} + E^{(1)} + E^{(2)} + \dots) (\ket{\psi^{(0)}} + \ket{\psi^{(1)}} + \ket{\psi^{(2)}} + \dots)
$$得到的各阶微扰本征方程为：
- 零阶：$\hat{H}_0 \ket{\psi^{(0)}} = E^{(0)} \ket{\psi^{(0)}}$；
- 一阶：$(\hat{H}_0 - E^{(0)}) \ket{\psi^{(1)}} = (E^{(1)} - \hat{H}') \ket{\psi^{(0)}}$；
- 二阶：$(\hat{H}_0 -E^{(0)}) \ket{\psi^{(2)}}  = (E^{(1)}- \hat{H}') \ket{\psi^{(1)}} + E^{(2)} \ket{\psi^{(0)}}$；
- 三阶：$(\hat{H}_0 -E^{(0)}) \ket{\psi^{(3)}}  = (E^{(1)}- \hat{H}') \ket{\psi^{(2)}} + E^{(2)} \ket{\psi^{(1)}}+E^{(3)}\ket{\psi^{(0)}}$；
- ……
上面的方程与$\bra{\psi^{(0)}}$作内积以后，就可以得到：
- 零阶：$E^{(0)}=\bra{\psi^{(0)}}\hat{H}'\ket{\psi^{(0)}}$；
- 一阶：$E^{(1)}=\bra{\psi^{(0)}}\hat{H}'\ket{\psi^{(1)}}$；
- 二阶：$E^{(2)}=\bra{\psi^{(0)}}\hat{H}'\ket{\psi^{(2)}}$；
- 三阶：$E^{(3)}=\bra{\psi^{(0)}}\hat{H}'\ket{\psi^{(3)}}$；
- ……
# 非简并态微扰论
不考虑微扰时，体系处于非简并态能级$E^{(0)}_{k}$，此时体系处于唯一波函数$\ket{\psi_{k}^{(0)}}$描述的状态。
微扰Hamilton矩阵元$$H_{mn}'=\bra{\psi_{m}^{(0)}}H'\ket{\psi_{n}^{(0)}}  $$
- 一阶近似公式：
	- 能量：$$E^{(1)}_{k}=\bra{\psi^{(0)}_{k}}\hat{H}'\ket{\psi^{(0)}_{k}}=H'_{kk}$$
	- 波函数：$$\ket{\psi_{k}^{(1)}} = \sum_{k',k'\neq k}\frac{-H'_{k'k}}{E^{(0)}_{k'}-E^{(0)}_k}|\psi_{k'}^{(0)}\rangle$$
- 二阶近似公式：
	- 能量：$$E^{(2)}_{k}=\langle \psi_{k}^{(0)}|\hat{H}'|\psi_{k}^{(1)}\rangle=\sum_{k',k'\neq k} \frac{-|H'_{k'k}|^2}{E^{(0)}_{k'} - E^{(0)}_k}$$
	- 波函数：$$\ket{\psi^{(2)}_{k}} =\sum_{k'',k''\neq k}\left\{ \left[ \sum_{k',k'\neq k} \frac{H'_{k''k'}H'_{k'k}}{(E_{k}^{(0)}-E_{k''}^{(0)})(E_{k}^{(0)}-E_{k'}^{(0)})} \right]- \frac{H'_{k''k}H'_{kk}}{(E_{k}^{(0)}-E_{k''}^{(0)})^2} \right\}\ket{\psi_{k''}^{(0)}} $$
- 收敛条件：$$\left| \frac{H'_{k'k}}{E_{k}^{(0)}-E_{k'}^{(0)}} \right|\ll 1$$应该尽量让收敛快一些，因为高阶的近似计算非常麻烦。
