# 平面
取定一个仿射标架$[O;\vec{d}_{1},\vec{d}_{2},\vec{d}_{3}]$，用一个点$M(x_{0},y_{0},z_{0})$和两个不共线向量$$\vec{\upsilon}_{1}=(X_{1},Y_{1},Z_{1}),\vec{\upsilon}_{2}=(X_{2},Y_{2},Z_{2})$$确定一个平面，于是该平面任意一点$M$有$$\overrightarrow{M_{0}M}=\lambda\vec{\upsilon}_{1}+\mu\vec{\upsilon}_{2}$$
平面方程：
- 参数式：$$\begin{equation}
        \left\{
	\begin{aligned}
		&x = x_{0} + \lambda X_{1} + \mu X_{2}\\
		&y = y_{0} + \lambda Y_{1} + \mu Y_{2}\\
		&z = z_{0} + \lambda Z_{1} + \mu Z_{2}\\
	\end{aligned}
	\right.\end{equation}$$
