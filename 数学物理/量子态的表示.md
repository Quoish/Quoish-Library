#量子力学 
# 量子力学的Hilbert空间
全体波函数构成的集合$\mathcal{H}$，若在其上定义内积为$$(\Phi,\Psi)=\int_全 [\Phi(\vec r,t)]^*\Psi(\vec r,t)\text{d}\tau$$其具有：
- 正定性：$(\Phi,\Psi)\geq0$。
- 共轭对称性/Hermite性：$(\Phi,\Psi)^*=(\Psi,\Phi)$。
- 对第一变量半线性：$(c_1\Phi_1+c_2\Phi_2,\Psi)=c_1^*(\Phi_1,\Psi)+c_2^*(\Phi_2,\Psi)$。
- 对第二变量线性性：$(\Phi,c_1\Psi_1+c_2\Psi_2)=c_1(\Phi,\Psi_1)+c_2(\Phi,\Psi_2)$。

在此内积下我们可以诱导正交的概念。
那这样的集合将会是一个**无穷维的复Hilbert空间**，这是量子力学的背景空间。
>注意：在数学中，Hilbert空间要求其具有完备性。大多数的波函数都具有平方可积性，根据Riesz-Fischer定理，平方可积函数构成的空间是完备的，因此可以自动具有完备性。但是描述量子态的波函数并不全是平方可积的，如动量的本征态函数，因此实际上量子力学所使用的是扩大了的广义Hilbert空间。

# 表象变换
## 波函数在表象中
若波函数以某一力学量算符对应的归一化正交函数系展开，则称该波函数处于该力学量的*表象*中。
对于某一力学量$A$，若我们求解本征方程$$\hat{A}\Psi=A\Psi$$非简并的情况下，我们将会得到一系列的本征函数$\{ \psi_{n} \}$，以及它们对应的不同的本征值$\{ A_{n} \}$，这些本征函数它们彼此正交。而在简并的情况下，虽然本征函数彼此不一定正交，但我们可以通过Schmit正交化化为彼此正交的。
根据Riesz-Fischer定理，我们可以将任意波函数$\Psi$以此本征函数为基底展开。
所以同一个波函数，我们可以透过诸多不同的表象去观察。
对于坐标表象来说，那就是$$\hat{r}\Psi(\vec{r},t)=\vec{r}\Psi(\vec{r},t)$$可见其实$\Psi(\vec{r},t)$这一关于波函数的“具象化”形式，只是在坐标表象下观察波函数时可以看到的现象罢了。
三维自由粒子在坐标表象下的波函数为$(2\pi \hbar)^{-3/2}e^{i(\vec{p}\cdot \vec{r}-Et)/\hbar}$，而在动量表象下就是$\delta(\vec{p}'-\vec{p})$。波函数的具体形式与表象的选取有着密切关联，这就类似于同一个向量在不同的坐标基下表现出不同的坐标。
## 力学量算符在表象中
设两个波函数，经算符$\hat A$作用后有$\Phi=\hat A \Psi$，及一个标准正交基函数$\eta_1,\eta_2,\cdots,\eta_n$，在此基底下（或该表象下）能展开为$$\Phi=\sum_i^{\infty} \varphi_i\eta_i,\Psi=\sum_i^{\infty}\psi_i\eta_i$$于是能分别用坐标$(\varphi_1,\varphi_2,\cdots,\varphi_n,\cdots),(\psi_1,\psi_2,\cdots,\psi_n,\cdots)$表示这两个波函数。
那么算符的作用就可以用矩阵表示，即$$\hat A
\left(
\begin{matrix}
\psi_{1},
\psi_{2},
\cdots,
\psi_{n},
\cdots
\end{matrix}
\right)
=
\left(
\begin{matrix}
\varphi_{1},
\varphi_{2},
\cdots,
\varphi_{n},
\cdots
\end{matrix}
\right)
\left(
\begin{matrix}
a_{11}&a_{12}&\cdots&a_{1s}&\cdots\\
a_{21}&a_{22}&\cdots&a_{2s}&\cdots\\
\vdots&\vdots&\vdots&\vdots&\ddots\\
a_{n1}&a_{21}&\cdots&a_{ns}&\cdots\\
\vdots&\vdots&\vdots&\vdots&\ddots\\
\end{matrix}
\right)
=
\left(
\begin{matrix}
\varphi_{1},
\varphi_{2},
\cdots,
\varphi_{n},
\cdots
\end{matrix}
\right)A$$这里的$A$就是力学量$A$在该表象下的矩阵表示。
矩阵元可以通过下式计算$$(\varphi_i,\hat A\psi_j)=a_{ij}$$
根据线性代数的知识可知，在不同基底下同一线性映射的矩阵表示酉相似，在量子力学中便是在不同表象下同一力学量算符的矩阵表示酉相似$$B=T^{-1}AT$$
## 量子力学与线性代数
这部分内容是线性代数在量子力学中的实际应用。下面总结了量子力学概念在线性代数中的对应关系：
- 力学量算符 → 线性映射（变换）
- 算符矩阵 → 线性映射在某基底下的矩阵表示
- 表象 → 某一标准正交基
- Hermite算符在某一表象下的矩阵是Hermite矩阵 → Hermite变换在标准正交基下的矩阵是Hermite矩阵
- 力学量算符在其自身表象下的矩阵是对角矩阵 → 线性映射在特征向量组成的基下是对角矩阵
- 一个表象到另一个表象的变换是酉变换 → 一个标准正交基到另一个标准正交基的过渡矩阵是酉矩阵

有了力学量算符的矩阵表示，求解本征方程的过程就变成了一个线性代数中的本征值问题：$$\hat{A} \Psi=\lambda\Psi\implies A\Psi=\lambda \Psi \implies |A-\lambda I|=0$$
# Dirac符号
Dirac符号是用于描述量子态的一种符号形式。
- Dirac右矢：表示量子态或本征函数
	- $|\Psi\rangle$表示用波函数$\Psi$描述的量子态。
	- $|x'\rangle$表示本征值为$x'$的位矢$x$分量本征函数。
	- $|p_x\rangle$表示本征值为$p_x$的动量$x$分量本征函数。
	- $|E_n\rangle$表示本征值为$E_n$的能量本征函数。
	- $|n\rangle$表示第$n$号离散本征值$a_n$的能量本征函数。

相比直接用坐标和时间表示的波函数$\Psi(\vec{r},t)$，Dirac符号所描述的波函数$\ket{\Psi}$是一种更加抽象的状态。前者是波函数在坐标表象下的投影，而后者并不依赖于具体的表象。
- Dirac左矢：表示由内积诱导的泛函
Dirac左矢是从Hilbert空间映射到复数域的一个泛函$$\langle\phi|\cdot\rangle:\mathcal{H}\to \mathbb{C},\forall \psi \in \mathcal{H},\langle \phi|\psi\rangle = (\phi,\psi)$$其中$\langle\phi|$是$\phi\in \mathcal{H}$所对应的在对偶空间的元素，这样的$\langle\phi|$组成的空间称作Dirac左矢空间，它是Hilbert空间的对偶空间。

不过，抛弃上面抽象的理解，我们还可以用具体的语言来解释Dirac左矢的意义。
