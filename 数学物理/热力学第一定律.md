#热力学 #物理化学 
# 内能
James Joule做过两个著名实验。
第一个是重物下降带动叶片在绝热水缸中的水搅动，水温受搅动而升温。
第二个是电阻放在绝热水缸中，通电使得水温升高。
结果发现，不同绝热过程使物体升高一定的温度，所做的功是相同的。
换句话说，系统经绝热过程，从初态到终态，过程中系统做的功*仅仅取决于系统的终末态，而与具体过程无关*。这个路径无关的性质说明这表明系统存在一个状态量（势函数）$U$，外界做的功仅取决于终末态的差：$$U_{末}-U_{初}=A$$
称状态函数$U$为*内能*。
上面看样子内能的定义只能精确到相差任意一个函数（势函数的性质大抵都如此），但在统计物理中，我们会得到内能明确的物理含义——系统微观粒子的能量总和，内能的零点就这样明确了。
# 热力学第一定律
如果系统经历的不是绝热过程，在过程中外界对系统做的功就不等于内能的变化了，二者的差就定义为系统在过程中以热量形式从外界吸取的能量$$Q=U_{末}-U_{初}-A$$热量$Q$的单位为Joule（$\mathrm{J}$）。
上式写成$$U_{末}-U_{初}=A+Q$$时可以发现，通过*做功和热传导*两种方式，都会转换为系统的内能。
写成微分形式就变为$$\mathrm{d}U=\mathrm{d}A+\mathrm{d}Q$$这被称作*热力学第一定律*。
>注意：$U$是态函数，所以$\mathrm{d}U$是全微分，而$A$和$Q$都只是变化量，$\mathrm{d}A,\mathrm{d}Q$都只是微小变化量，而不是全微分。

Joule还做过气体在真空中绝热膨胀时的过程。因为真空中外界压强为0，所以外界无做功，而且因为是绝热过程，气体也不会传递热。所以气体膨胀的过程中内能不会发生变化，Joule在实验中还发现温度也没有变化，但是体积却变了。
这说明：*（对于理想气体来说），内能不是体积的函数，只是体积的函数*。
后来人们发现对实际气体来说，上述结论只有在$P_{0}\to 0$，也就是接近理想气体的时候才能成立，而实际气体的内能有时也会是体积的函数。
# 热容
我们用热容来描述物体传热或储热的能力。
*热容*定义为$$C=\lim_{ \Delta T \to 0 } \frac{\Delta Q}{\Delta T}$$
平常使用的热容通常的定压或者定容过程的热容，分别称为*定压热容*和*定容热容*。
$$C_{V}=\lim_{ \Delta T \to 0 } \left( \frac{\Delta Q}{\Delta T} \right)_{V} $$$$C_{P}=\lim_{ \Delta T \to 0 } \left( \frac{\Delta Q}{\Delta T} \right)_{P} $$
## 迈耶公式
当体积不变时，外界不做功，按热力学第一定律有$\Delta Q=\Delta U$，于是$$C_{V}=\lim_{ \Delta T \to 0 } \left( \frac{\Delta U}{\Delta T} \right)_{V}=\left( \frac{ \partial  U }{ \partial  T}  \right)_{V}$$
当压强不变时，外界做的功$A=-P\Delta V$，按热力学第一定律有$\Delta Q=\Delta U+P\Delta V$，于是$$C_{P}=\lim_{ \Delta T \to 0 } \left( \frac{\Delta U+P\Delta V}{\Delta T} \right)_{P}=\left( \frac{ \partial  U }{ \partial  T}  \right)_{P}+P\left( \frac{ \partial  V }{ \partial  T}  \right)_{P}$$
我们考虑理想气体的情况，所以$$\left( \frac{ \partial  U }{ \partial  T}  \right)_{V}=\left( \frac{ \partial  U }{ \partial  T}  \right)_{P}=\frac{ \mathrm{d}  U }{ \mathrm{d}  T},\quad p\left( \frac{ \partial  V }{ \partial  T}  \right)_{P}= R$$所以我们可以得到以下关系式：$$C_{P}-C_{V}=R$$这被称作*Meyer公式*。
定义$\gamma=C_{P}/C_{V}$，这个参量称为比热容比或*绝热系数*。
## 比热容
由于热容是广延量，我们定义强度量比热容$$c=\frac{C}{m}$$其中$m$是物体的质量。
# 摩尔比热容
有的时候质量在一些领域比如化学反应中使用并不方便，于是我们定义摩尔比热容$$c_{m}=\frac{C}{\nu}$$其中$\nu$是物体的物质的量。
