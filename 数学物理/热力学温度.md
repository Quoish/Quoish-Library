#热力学 #物理化学 
# 温度
温度表征的是物体的冷热程度。
不同的物体在各种不同情形下的温度可以通过量化来比较是基于上面说到的一个事实：即两个物体接触以后在足够长的时间内必将达到一个平衡态。
当两个物体已达到热平衡以后，我们的直觉认为这两个物体一定是同样的热冷，因之温度相等，由此得到一个重要的结论：
>两个互为热平衡的物体，其温度相等。

这个结论给予温度一个定性的定义，把一个物体的温度联系到另一个物体，这第二个物体可以选为温度计。不过，这个定义依然是定性的，没有数学表示。
从平衡态的性质可以得出无论多少个物体互相接触都能达到平衡，而且有：
>**热力学第零定律**：
>假定某甲物体分别与某乙和某丙达到平衡，那么如果让乙与丙接触，它们一定是互为平衡而不发生新的变化。

这指出了温度的一个最基本的性质：*一切互为热平衡的物体有相同的温度*。
# 温标
为了确定温度的数值，我们必须选定一种物质作为温度计，对这个物质的冷热状态给一个数值的表示法。由于不同冷热状态描写这个物体的各类变数的数值各不相同，温度必定会和其中的一个或几个发生关系。通常是选择一种随冷热状态显著变化的物理量作为温度的标志，最常用的是*热膨胀现象下的体积*。
温度的数值表示法被称作*温标*。近代最常用的温标是摄氏度（℃），它规定一个标准大气压下纯水的冰点记为0℃，纯水的沸点记为100℃。
# 理想气体温标
规定0℃与100℃之后，其他的温度由作为温度标志的物理量的线性函数确定。但由于所选温度计物质的性质不同，所选物理量的不同，除了0℃和100℃，其他温度都会有细微的差别。
当我们用定容气体温度计时：定容气体温度计的压强保持不变，而用气体体积的改变作为温度的标志，这样的温标记为$t_{p}$，其与气体体积的关系是$$\frac{t_{p}}{100}=\frac{V_{t_{p}}-V_{冰点}}{V_{沸点}-V_{冰点}}$$
当我们用定压气体温度计时：定压气体温度计的体积保持不变，而用气体压强的改变作为温度的标志，这样的温标记为$t_{v}$，其与气体压强的关系是$$\frac{t_{v}}{100}=\frac{p_{t_{p}}-p_{冰点}}{p_{沸点}-p_{冰点}}$$
实验证明：各种不同气体的$t_{p},t_{v}$几乎相等，只有细微的差别。当压强极低的时候，这样的差别近乎消失，也就是$$\lim_{ p \to 0 }t_{p}=\lim_{ p \to 0 } t_{v}=t $$
压强很低的时候就相当于气体无限稀薄，气体分子之间的相互作用可以不计了，也就是相当于理想气体。所以称这个极限$t$为*理想气体温标*。
热力学中采用的是绝对温标，用$T$表示，单位开尔文（$\mathrm{K}$），其与摄氏温标$t$的关系为$$T=t+T_{0}$$其中$T_{0}=273.15$。而$T=0\mathrm{K}$或者说$t=-273.15\text{℃}$的温度称为*绝对零度*。
用任意一种测温物质，根据任意一种定温度数值的规律所定的一种温标称为*经验温标*记为$\vartheta$。$\vartheta$可以是$t_{v}$，可以是$t_{p}$，可以是$t$，可以是$T$，也可以是任何一种温度计所指示的温度。
