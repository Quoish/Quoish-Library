# 开式光腔的作用
1. 抑制模式数。
2. 提供正反馈。
3. 产生损耗。
4. 滤波。
5. 决定光束特性。
6. 决定激光稳定震荡性。
# 光腔的损耗
## 引起损耗的原因
1. 非轴向传播的几何偏折逸出腔外。
2. 输出镜$M_{2}$的反射率$R_{2}<1$导致一部分光透射出去（必须有）。
3. $M_{1},M_{2}$镜内介质不理想引起的吸收、散射。
4. $M_{1},M_{2}$镜尺寸有限导致的衍射。
## 损耗的描述方法
### 平均单轮损耗因子$\delta$
从$M_{1}$出发、$M_{2}$反射、$M_{1}$反射称为一个往返。（注意需要在$M_{1}$再反射一次）
一个往返后光强从$I_{0}$变为$I_{1}$，可以描述为$$I_{1}=I_{0}e^{ -2\delta }$$
$\delta$称为平均单轮损耗因子。
### 腔内光子平均寿命$\tau_{R}$
称$I_{0}$下降到$1/e$的时间为光子的寿命$\tau_{R}$。可以推导出$$I=I_{0}e^{ -t/\tau_{R} }$$其中$$\tau_{R}=\frac{L}{c\delta}$$$L$是腔长。
### 光腔品质因子$Q$
与电子学的定义一致$$Q=2\pi \nu \frac{E(t时刻储存的能量)}{P(损耗的功率)}=\omega \tau_{R}$$
### 光腔本征线宽$\Delta \nu_{e}$
线型函数$g(\nu ,\nu_{0})$的半高全宽称为光腔本征线宽$\Delta \nu_{e}$。可以推导出$$\Delta \nu_{e}=\frac{1}{2\pi \tau_{R}}$$
