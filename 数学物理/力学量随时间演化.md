# 守恒量
## Ehrenfest关系
量子力学中，**力学量的期望**不随时间变化的力学量被称为守恒量。
为了判断什么量能守恒，我们要研究力学量随时间的变化。设力学量$\hat{F}$的期望为$\bar{F}$，对$\bar{F}$求时间$t$的导数，根据[[Schrödinger方程]]，可以得到如下关系式$$\boxed{\frac{\text{d}\bar{F} }{\text{d}t } =\frac{\partial \bar{F}}{\partial t}+\frac{1}{i\hbar} \overline{[\hat F,\hat H]} }$$这被称为*Ehrenfest关系*。
力学量不显含时的情况，就可化简为$$\frac{ \mathrm{d}  \bar{F} }{ \mathrm{d}  t} = \frac{1}{i\hbar} \overline{[\hat F,\hat H]} $$
可见力学量守恒$\dfrac{ \mathrm{d}  \bar{F} }{ \mathrm{d}  t}=0$需要满足两大条件：
- 不显含时（第一项为零）。
- 与Hamilton算符对易（第二项为零）。

以下是一些是守恒的例子：
- 自由粒子动量守恒$$\hat H=\frac{1}{2m}\hat p^2 \implies\frac{\text{d}\bar{p} }{\text{d}t } =\frac{1}{i\hbar} \overline{[\hat p,\hat H]} =0$$
- 中心力场的角动量守恒$$\hat H=-\frac{\hbar^2}{2m}\frac{1}{r^2}\frac{\partial }{\partial r} \left ( r^2\frac{\partial }{\partial r}  \right )  +\frac{\hat L^2}{2mr^2} + U(r)\implies \frac{\text{d}\bar{L^2} }{\text{d}t } =\frac{1}{i\hbar} \overline{[\hat L^2,\hat H]} =0 $$
- Hamilton不含时体系能量守恒$$\frac{\partial \bar{H}}{\partial t}=0 \implies \frac{\text{d}\bar{H} }{\text{d}t } =\frac{1}{i\hbar} \overline{[\hat H,\hat H]} =0$$
- Hamilton反演不变体系宇称守恒$$\hat P\hat H=\hat H \implies \frac{\text{d}\bar{P} }{\text{d}t } =\frac{1}{i\hbar} \overline{[\hat P,\hat H]} =0$$

>需要注意的是，与经典力学不同，量子力学的守恒量并不代表体系中这个量始终取确定的值，而只是概率分布和期望是确定的值罢了。并且，守恒量并不一定可以同时取确定的值，譬如中心力场中的粒子，角动量三个方向的分量都是守恒的，但是它们彼此不对易，所以彼此不能同时确定。

## 守恒量与简并度
- 设体系中有两个守恒量$\hat{F},\hat{G}$彼此不对易，即$[\hat{F},\hat{H}]=0,[\hat{G},\hat{H}]=0$，但$[\hat{F},\hat{G}]\neq 0$，则体系能级一般是简并的（除了极少数例外）。
证：
假设能级$E_{n}$不是简并的，则存在唯一的$\psi_{n}$使得$\hat{H}\psi_{n}=E_{n}\psi_{n}$，而$\hat{H}$与$\hat{F},\hat{G}$都对易，这意味着$\psi_{n}$同时是$\hat{H}$和$\hat{F}$，$\hat{H}$和$\hat{G}$的共同本征态（见共同本征态的判定），所以也是$\hat{F},\hat{G}$的共同本征态，假设$\hat{F},\hat{G}$对应的本征值为$F',G'$，这意味着$$[\hat{F},\hat{G}]\psi_{n}=(F'G'-G'F')\psi_{n}=0$$
在物理上，这是一个极为苛刻的条件，绝大多数情况下都是不可能的，所以能级只能是简并的。
一些例外情况如因为$[\hat{l}_{x},\hat{l}_{y}]=i\hbar \hat{l}_{z}$，如果$[\hat{l}_{x},\hat{l}_{y}]\psi=0$，也就是三个角动量分量同时为0的时候这是可以简并的。
- $\hat{F}$是体系中的守恒量，如果体系中某条能级$E_{n}$不简并，即对于该本征值$E_{n}$只有一个本征态$\psi_{n}$，则$\psi_{n}$也是$\hat{F}$的本征态。
- 设体系中有两个守恒量$\hat{F},\hat{G}$，满足$[\hat{F},\hat{G}]= C$，其中$C$是非零常数，则体系的所有能级都简并，且简并度为无穷大。
证：
上面的命题已经说明，如果要简并，那必须$[\hat{F},\hat{G}]\psi_{n}=0$，但在$[\hat{F},\hat{G}]= C$时这是不可能的，所以必定都简并。
设能级的简并度为$g_{n}$，对应个$g_{n}$个本征态函数为$\psi_{n\nu},\nu=1,2,\dots,g_{n}$。以$\{ \psi_{n\nu} \}_{\nu=1}^{g_{n}}$为基矢，量算符的矩阵分别为$F,G$，求$FG$和$GF$的迹。
若$g_{n}$有限，则必有$$\mathrm{tr}(FG)=\mathrm{tr}(GF)\implies \mathrm{tr}([F,G])=0$$但是$$\mathrm{tr}([F,G])=Cg_{n}$$所以矛盾。$g_{n}$必定为无限值。
# Ehrenfest定理
这一节我们讨论量子力学到经典力学的过渡。
设粒子质量为$m$，我们将粒子视为概率波的波包，而波包中心为位矢的平均值$\bar{\vec{r}}$。注意，在研究量子力学到经典的过渡时，把粒子视为概率波包这不同与疏密波说认为的那样把粒子完全视为波包，概率波包的扩散并不会导致粒子变得裂开，而只是出现位置的不确定性在扩散。
波包在势场$V(\vec{r})$中运动，体系的Hamilton量为$$H=\frac{p^2}{2m}+V(\vec{r})$$它和位矢、动量的对易子分别为（因为算符和自身函数对易）$$[\hat{r},\hat{H}]=\left[ \hat{r}, \frac{\hat{p}^2}{2m} \right]=\frac{i\hbar}{m} \hat{p},\quad [\hat{p},\hat{H}]=[\hat{p},V(\vec{r})]=-i\hbar \nabla V(\vec{r})$$对$\hat{r},\hat{p}$应用Ehrenfest关系$$\frac{ \mathrm{d}  \bar{\vec{r}} }{ \mathrm{d}  t} =\frac{1}{i\hbar}\overline{[\hat{r},\hat{H}]}=\frac{\bar{\vec{p}}}{m},\quad \frac{ \mathrm{d}  \bar{\vec{p}} }{ \mathrm{d}  t} =\frac{1}{i\hbar}\overline{[\hat{p},\hat{H}]}=-\overline{\nabla V(\vec{r})}$$这与经典粒子的正则方程类似。
联立可得$$\boxed{m\frac{ \mathrm{d}  ^2 }{ \mathrm{d}  t^2} \bar{\vec{r}}=\overline{\vec{F}(\vec{r})}}$$此之谓*Ehrenfest定理*。它与Newton第二定律的形式很相似，体现了量子力学在平均值意义上可以过渡到经典力学。
不过要获得与Newton第二定律完全一致的结果，需要有近似$$\overline{\vec{F}(\vec{r})}\approx\vec{F}(\bar{\vec{r}})$$下面研究这种近似在什么情况下才能成立，为简单起见，我们只研究一维的情况。
将$F(x)$以$\bar{x}$为中心进行Taylor展开$$F(x)=F(\bar{x})+F'(\bar{x})(x-\bar{x})+\frac{1}{2!}F''(\bar{x})(x-\bar{x})^2+\dots$$两边取期望，由于$\langle x-\bar{x} \rangle=0$，所以$$\langle F(x) \rangle =F(\bar{x})+0+\frac{1}{2!}F''(\bar{x})(\Delta x)^2+\dots$$其中$(\Delta x)^2=\langle (x-\bar{x})^2 \rangle$是$x$的涨落/方差。
所以如果要想实现上面的近似，需要有$$|F(\bar{x})|\gg \left| \frac{1}{2}F''(\bar{x})(\Delta x)^2 \right| $$这就意味着两件事：
1. 波包很窄，而且在运动中扩散地不厉害（$\Delta x$小）；
2. $V(x)$在空间中变化缓慢。如果$V(x)$是小于三次的多项式，那$F''(x)=\dfrac{ \mathrm{d}  ^3 V}{ \mathrm{d}  x^3}=0$，将自动满足条件。所以自由粒子、线性势、谐振子势中窄波包的运动和经典粒子非常相似。
# 含时演化的三种图像
## Schrödinger图像
迄今为止，我们认为态（波函数）是随着时间变化的，而力学量（算符）本身是不随时间变化的，这种量子力学的图景被称为Schrödinger图像（Picture）。
下面我们假设**Hamilton量不含时**。
Schrödinger方程为$$i\hbar \frac{ \partial   }{ \partial  t} \psi(t)=\hat{H}\psi(t)$$若令$$\psi(t)=U(t,0)\psi(0)$$其中$U(t,0)$是时间演化算符，描绘了态从$t=0$到$t=t$时的变化。为了保证概率守恒$(\psi(t),\psi(t))=(\psi(0),\psi(0))$，就必须要求$U$是酉变换。
代入Schrödinger方程，可以解得$$U(t,0)=e^{ -iHt/\hbar }$$
