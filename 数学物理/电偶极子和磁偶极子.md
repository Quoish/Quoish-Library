#电磁学 
# 电偶极子
一个电偶极子由一对等量异号的电荷 $±q$ 相隔一个微小距离 $l$ 构成。
通常在远场近似条件（$r\gg l$）下讨论电偶极子的力学性质。
电偶极子用电偶极矩描述： $$\vec{p} = q\vec{l}$$ (其中 $\vec{l}$ 是从负电荷指向正电荷的位移矢量)
## 作为主体
1. 电势
	中心处于$\vec{r}'$的电偶极子$\vec{p}$，在$\vec{r}$产生的电势为$$u(\vec{r})=-\frac{1}{4\pi\varepsilon_{0}} \frac{\vec{p}\cdot (\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3}$$
2. 电场
	由电势求梯度可以得到
$$\vec{E}=\frac{1}{4\pi \varepsilon_{0}}\frac{2ql\cos \theta}{r^3}\hat{r}+\frac{1}{4\pi \varepsilon_{0}}\frac{ql\sin \theta}{r^3}\hat{\theta}$$
## 作为客体
1.  力矩
    在均匀电场$\vec{E}$ 中，电偶极子会受到一个力矩$\tau$，使其倾向于转向与外电场方向一致。
    公式： $\tau=\vec{p}\times \vec{E}$
2.  势能
    电偶极子在电场中具有一定的势能。势能最低的状态是当 $\vec{p}$ 与 $\vec{E}$ 方向一致时。
    公式： $U = -\vec{p}\cdot \vec{E}$
    *   当 $θ = 0°$ (平行)，$U = -pE$ (能量最低，稳定平衡)
    *   当 $θ = 90°$， U = 0
    *   当 $θ = 180°$ (反平行)，$U = +pE$ (能量最高，不稳定平衡)
3.  净力
    在均匀电场中，作用在正负电荷上的力大小相等、方向相反，因此**净力为零**。
    公式： $\vec{F}=0$
    在**非均匀电场**中，偶极子会受到一个净力。这个力指向电场强度增加的方向（如果偶极矩的方向与电场方向一致）。
    近似公式（对于小偶极子或缓变电场）： $\vec{F}=(\vec{p}\cdot \nabla)\vec{E}$
    *   这个公式是梯度算符 ($\vec{p}· ∇$) 作用于电场矢量 $\vec{E}$ 的结果。它意味着力的大小取决于电场在偶极子方向上的变化率。
    *   在电场更强的区域，偶极子的势能更低，因此偶极子会被吸引到电场更强的区域。
## 双电偶极子
设真空有两个电偶极子$\vec{p}_{1},\vec{p}_{2}$，$\vec{p}_{1}$产生的电场为$$\vec{E}=-\frac{1}{4\pi\varepsilon_{0}}\nabla\left( \frac{\vec{p}_{1}\cdot \vec{r}}{r^3} \right)=\frac{1}{4\pi \varepsilon_{0}}\frac{3(\vec{p}_{1}\cdot \hat{r})\hat{r}-\vec{p}_{1}}{r^3}$$
在$\vec{p}_{1}$产生的电场里，$\vec{p}_{2}$的势能即为$$U(r)=-\vec{p}_{2}\cdot \vec{E}= \frac{1}{4\pi\varepsilon_{0}} \frac{\vec{p}_{1}\cdot \vec{p}_{2}-3(\vec{p}_{1}\cdot \hat{r})(\vec{p}_{2}\cdot \hat{r})}{r^3}$$
# 磁偶极子
一个磁偶极子可以是一个小电流环，或者一个基本粒子（如电子）的固有磁矩。
对于面积为 $S$，电流为 $I$ 的小电流环， $\vec{m} = I\vec{S}$ （其中 $\vec{S}$ 的方向由右手螺旋确定）
## 作为主体
1. 磁矢势
	中心处于$\vec{r}'$的磁偶极子$\vec{m}$，在$\vec{r}$产生的磁矢势为$$A(\vec{r})=\frac{\mu_{0}}{4\pi} \frac{\vec{m}\times (\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3}$$
2. 磁场
	对上面取旋度就可以得到$$\vec{B}=\frac{\mu_{0}}{4\pi} \frac{2m\cos \theta }{r^3}\hat{r}+ \frac{\mu_{0}}{4\pi} \frac{m\sin \theta}{r^3}\hat{\theta}$$
## 作为客体
1.  力矩
    在均匀磁场 $\vec{B}$ 中，磁偶极子会受到一个力矩$\tau$，使其倾向于转向与外磁场方向一致
    公式：$\tau=\vec{m}\times \vec{B}$
2.  势能
    磁偶极子在磁场中具有一定的势能。
    公式： $U = -\vec{m} · \vec{B}$
    *   当 $θ = 0°$ (平行)，$U = -mB$ (能量最低，稳定平衡)
    *   当 $θ = 90°$， $U = 0$
    *   当 $θ = 180°$ (反平行)，$U = +mB$ (能量最高，不稳定平衡)
3.  净力
    在均匀磁场中，作用在电流环上的安培力的矢量和为零，因此净力为零。
    公式： $\vec{F}=0$
    在**非均匀磁场**中，磁偶极子会受到一个净力。这是许多磁学现象（如铁屑被磁铁吸引）的基础。
    近似公式（对于小偶极子或缓变磁场）：$\vec{F} = ∇(\vec{m} · \vec{B})$
    *   这个公式是磁偶极子势能 U 的负梯度： $\vec{F} = -∇U = ∇(\vec{m} · \vec{B})$
    *   与电偶极子类似，如果 $\vec{m}$ 与 $\vec{B}$ 平行，力会指向磁场更强的区域。
