#经典力学 
# 守恒量
在力学系统运动过程中，描述其状态的$2s$个变量$q_i,\dot{q}_i(i=1,2,...,s)$随时间变化，但是存在关于这些变量的某些函数，其值在运动过程中**不随时间变化，且仅由初始条件决定**，这样的函数称为*运动积分*（也称为首次积分）。
之所以称为“运动积分”，是因为它通常来源于对运动微分方程的一次积分。在求解微分方程进行一次积分的过程中，一般都会根据初值条件确定一个常数，这个常数指的就是运动积分。
**具有$s$个自由度的运动系统，独立的运动积分数量只有$2s-1$个**，从下面的论述中可以看出。
如果我们解出体系运动方程，那将获得以下$2s$个关于时间$t$和$2s$个常量的函数$$q_{i}=q_{i}(t,C_{1},C_{2},\dots,C_{2s}),\quad \dot{q}_{i}=\dot{q}_{i}(t,C_{1},C_{2},\dots,C_{2s}),\quad i=1,2,\dots,s$$然而，运动积分是与$t$无关的，这就要求我们消去$t$，我们使用$2s$个方程最多能得到$2s-1$个不显含$t$的函数关系。
然而，并不是所有的运动积分在力学中有相同的重要性，其中一些运动积分源自时间和空间的均匀性和各向同性这样的基本性质，这种运动积分的恒定不变性才具有深刻的意义，由这些运动积分表示的量才是*守恒量*，它们具有一个重要的共同性质——**可加性**。对于几个相互独立部分组成的系统，守恒量的值等于各个部分相应值之和。
# 时间均匀性-能量守恒
时间具有均匀性意味着Lagrange函数不显含时，也就是$\dfrac{ \partial  \mathcal{L} }{ \partial  t}=0$，因此Lagrange函数$\mathcal{L}(q,\dot{q},t)$对时间的全导数可以写作$$\frac{ \mathrm{d}  \mathcal{L} }{ \mathrm{d}  t} =\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  q_{i}} \dot{q}_{i}+\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \dot{q}_{i}}q_{i}$$利用Lagrange方程替换$\dfrac{ \partial  \mathcal{L} }{ \partial  q_{i}}$，变为$$\frac{ \mathrm{d}  \mathcal{L} }{ \mathrm{d}  t} =\sum_{i}q_{i}\frac{ \mathrm{d}   }{ \mathrm{d}  t} \frac{ \partial  \mathcal{L} }{ \partial  q_{i}} +\sum_{i}\frac{ \mathrm{d}  q_{i} }{ \mathrm{d}  t} \frac{ \partial  \mathcal{L} }{ \partial  \dot{q}_{i}} =\sum_{i}\frac{ \mathrm{d}   }{ \mathrm{d}  t} \left( q_{i}\frac{ \partial  \mathcal{L} }{ \partial  q_{i}}  \right)$$所以$$\frac{ \mathrm{d}   }{ \mathrm{d}  t} \left( \sum_{i}q_{i}\frac{ \partial  \mathcal{L} }{ \partial  q_{i}}  -\mathcal{L}\right)=0$$这是个守恒量，定义为$$E:=\sum_{i}q_{i}\frac{ \partial  \mathcal{L} }{ \partial  q_{i}}-\mathcal{L} $$这就是*能量*。由Lagrange函数的可加性可以直接得到能量的可加性。
$\mathcal{L}=T-U$，由齐次函数的Euler定理可以得到$$\sum_{i}\dot{q}_{i}\frac{ \partial  T }{ \partial  \dot{q}_{i}} =2T$$所以$$E=T+U$$
# 空间均匀性-动量守恒
空间均匀性意味着经过平移Lagrange量$\mathcal{L}$保持不变，即对于从$\vec{r}$到$\vec{r}+\vec{\varepsilon}$的任意位移$\vec{\varepsilon}$$$\delta \mathcal{L}=\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \vec{r}_{i}}\cdot \delta \vec{r}_{i}:=\vec{\varepsilon}\cdot \sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \vec{r}_{i}}=0 $$它要对任意$\vec{\varepsilon}$成立，所以$$\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \vec{r}_{i}} =0$$这个等式若代入$\dfrac{ \partial  \mathcal{L} }{ \partial  \vec{r}_{i}}=\dfrac{ \partial  U }{ \partial  \vec{r}_{i}}=-\vec{F}_{i}$，就变成$$\sum_{i}\vec{F}_{i}=0$$这就是作用力与反作用力相等定律或*Newton第三定律*。
根据Lagrange方程在直角坐标系下的形式，上面的式子代换为$$\sum_{i}\frac{ \mathrm{d}   }{ \mathrm{d}  t} \frac{ \partial  \mathcal{L} }{ \partial  \vec{v}_{i}}=\frac{ \mathrm{d}   }{ \mathrm{d}  t}\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \vec{v}_{i}}   =0$$这是个守恒量，定义为$$\vec{P}:=\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  \vec{v}_{i}} $$这就是*动量*。可加性是显然的。
直角坐标系下有$$\vec{p}_{i}=m_{i}\vec{v}_{i}$$
若选取为广义坐标$$\vec{p}_{i}=\frac{ \partial  \mathcal{L} }{ \partial  \dot{q}_{i}} $$这就是*广义动量*。
Lagrange方程可以写成广义动量和广义力的关系$$\dot{\vec{p}}_{i}=\vec{F}_{i}$$
# 空间各向同性-角动量守恒
空间各向同性意味着封闭系统整体在空间中任意转动后，力学性质保持不变。
![[角动量守恒.png]]
经过在无穷小转动$\delta \vec{\varphi}$后，位矢变化$$\delta \vec{r}=\delta \vec{\varphi}\times \vec{r}$$速度变化$$\delta \vec{v}=\delta \vec{\varphi}\times \vec{v}$$最终对Lagrange量造成的变化为$$
\begin{equation}
\begin{aligned}
\delta \mathcal{L}&=\sum_{i}\left( \frac{ \partial  \mathcal{L} }{ \partial  \vec{r}_{i}}\delta \vec{r}_{i}+\frac{ \partial  \mathcal{L} }{ \partial  \vec{v}_{i}} \delta v_{i}  \right)\\
&=\sum_{i}[\dot{p}_{i}\cdot(\delta \vec{\varphi}\times \vec{r})+\vec{p}_{i}\cdot(\delta \vec{\varphi}\times \vec{v}_{i})]\\
&=\delta \vec{\varphi}\cdot\sum_{i}(\vec{r}_{i}\times \dot{\vec{p}}_{i}+\vec{v}_{i}\times \vec{p}_{i})\\
&=\delta \vec{\varphi}\cdot \frac{ \mathrm{d}   }{ \mathrm{d}  t} \sum_{i} (\vec{r}_{i}\times \vec{p}_{i})
\end{aligned}
\end{equation}
$$由$\delta \vec{\varphi}$的任意性可得$$\frac{ \mathrm{d}   }{ \mathrm{d}  t} \sum_{i}(\vec{r}_{i}\times \vec{p}_{i})=0$$这是个守恒量，定义为$$L:=\sum_{i}(\vec{r}_{i}\times \vec{p}_{i})$$这就是*角动量*。可加性也是显然的。
# 质心
质心是用来描述多质点系统运动的总体位置，定义为$$\vec{R}=\frac{\sum_{i}m_{i}\vec{r}_{i}}{\sum_{i}m_{i}}$$整体速度为$$\vec{V}=\frac{\sum_{i}m_{i}\vec{v}_{i}}{\sum_{i}m_{i}}$$整体质量为$$\mu=\sum_{i}m_{i}$$
若参考系$K'$相对于参考系$K$以$\vec{V}$运动，则$$\vec{P}=\vec{P}'+\mu \vec{V}$$$$E=E'+\vec{V}\cdot \vec{P}'+\frac{\mu V^2}{2}$$$$\vec{L}=L'+\mu \vec{R}\times \vec{V}$$
# 力学相似性
Lagrange量乘以任意常数不会改变运动方程。在一些重要情况下，利用这一点，无需实际求解运动方程就可以得到有关运动性质的一些有用的结论。
比如说势能是坐标的齐次函数的情况，即势能函数满足条件$$U(\alpha r_{1},\alpha r_{2},\dots,\alpha r_{s})=\alpha^kU(r_{1},r_{2},\dots,r_{s})$$其中$\alpha$是任意常数，$k$是势能函数是齐次次数。
若引入变换，让坐标变成$\alpha$倍，时间变成$\beta$倍，$$r_{i}\to \alpha r_{i},\quad t_{i}\to \beta t_{i}$$则所有速度$v_{i}$变成$\alpha/\beta$倍，动能变成$\alpha^2/\beta^2$倍，势能变成$\alpha^k$倍。
假如动能和势能变大的倍数相等，则变换的结果就是Lagrange量乘以常数，不改变运动方程，此时满足$$\frac{\alpha^2}{\beta^2}=\alpha^k\implies \beta=\alpha^{1-k/2}$$
这样一来，所有质点的坐标改变相同的倍数，意味着变换前后的运动轨迹几何上相似，仅仅是尺寸不同。
于是我们可以得出结论，**如果系统的势能是（直角）坐标的$k$次齐次函数，则由运动方程可以得到一系列几何上相似的不同轨迹，并且（不同轨迹上的相应点的)运动时间之比满足关系式：**$$\frac{t'}{t}=\left( \frac{l'}{l} \right)^{1-k/2}$$
相应地还有$$\frac{v'}{v}=\left( \frac{l'}{l} \right)^{k/2},\quad \frac{E'}{E}=\left( \frac{l'}{l} \right)^k,\quad \frac{L'}{L}=\left( \frac{l'}{l} \right)^{1+k/2}$$
对于均匀力场，$k=1$，所以$$\frac{t'}{t}=\left( \frac{l'}{l} \right)^{1/2}$$这就是自由落体运动的规律。
对于Newton万有引力和Coulomb力，$k=-1$，所以$$\frac{t'}{t}=\left( \frac{l'}{l} \right)^{3/2}$$这就是*Kepler第三定律*。
如果力学系统在有限空间中运动，势能是坐标的$k$次齐次函数，则动能和势能的时间平均值之间满足$$2\bar{T}=k\bar{U}$$这个关系式称为*位力定理*（Virial定理）。

---
证明：
因为动能是速度的2次齐次函数，根据齐次函数的Euler定理$$\sum_{i}\frac{ \partial  T }{ \partial  \vec{v}_{i}} \cdot \vec{v}_{i}=2T$$利用$\dfrac{ \partial  T }{ \partial  \vec{v}_{i}}=\vec{p}_{i}$，改写成$$2T=\sum_{i}\vec{p}_{i}\cdot \vec{v}_{i}=\frac{ \mathrm{d}   }{ \mathrm{d}  t} \sum_{i}\vec{p}_{i}\cdot \vec{r}_{i}-\sum_{i}\dot{\vec{p}}_{i}\cdot \vec{r}_{i}$$
我们将上面这个等式对时间平均，可以证明，如果函数$f(t)$是某个有界函数$F(t)$对时间的全导数，则$f(t)$对时间平均为零，因为$$\bar{f}=\lim_{ \tau \to \infty } \frac{1}{\tau}\int_{0}^{\tau}\frac{ \mathrm{d}   F}{ \mathrm{d}  t} \mathrm{d}t=\lim_{ \tau \to \infty } \frac{F(\tau)-F(0)}{\tau}=0$$
