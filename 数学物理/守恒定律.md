# 守恒量
在力学系统运动过程中，描述其状态的$2s$个变量$q_i,\dot{q}_i(i=1,2,...,s)$随时间变化，但是存在关于这些变量的某些函数，其值在运动过程中**不随时间变化，且仅由初始条件决定**，这样的函数称为*运动积分*（也称为首次积分）。
之所以称为“运动积分”，是因为它通常来源于对运动微分方程的一次积分。在求解微分方程进行一次积分的过程中，一般都会根据初值条件确定一个常数，这个常数指的就是运动积分。
**具有$s$个自由度的运动系统，独立的运动积分数量只有$2s-1$个**，从下面的论述中可以看出。
如果我们解出体系运动方程，那将获得以下$2s$个关于时间$t$和$2s$个常量的函数$$q_{i}=q_{i}(t,C_{1},C_{2},\dots,C_{2s}),\quad \dot{q}_{i}=\dot{q}_{i}(t,C_{1},C_{2},\dots,C_{2s}),\quad i=1,2,\dots,s$$然而，运动积分是与$t$无关的，这就要求我们消去$t$，我们使用$2s$个方程最多能得到$2s-1$个不显含$t$的函数关系。
然而，并不是所有的运动积分在力学中有相同的重要性，其中一些运动积分源自时间和空间的均匀性和各向同性这样的基本性质，这种运动积分的恒定不变性才具有深刻的意义，由这些运动积分表示的量才是*守恒量*，它们具有一个重要的共同性质——**可加性**。对于几个相互独立部分组成的系统，守恒量的值等于各个部分相应值之和。
# 时间均匀性-能量守恒
时间具有均匀性意味着Lagrange函数不显含时，因此Lagrange函数$\mathcal{L}(q,\dot{q},t)$对时间的全导数可以写作$$\frac{ \mathrm{d}  \mathcal{L} }{ \mathrm{d}  t} =\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  q_{i}} \dot{q}_{i}+\sum_{i}\frac{ \partial  \mathcal{L} }{ \partial  q} $$
