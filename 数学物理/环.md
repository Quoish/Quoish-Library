# 环的定义
若非空集合$R$上定义了两种运算$+,\cdot$，它对两运算**封闭**，并满足六条运算法则：
1. 加法结合律：$\forall a,b,c\in R$，有$a+(b+ c)=(a+ b)+ c$；
2. 加法交换律：$\forall a,b \in R$，有$a+b=b+a$；
3. 零元：$\forall a\in R$，$\exists o \in R$，使得$a+o=o +a=a$；
4. 负元：$\forall a\in R$，$\exists r\in R$，使得$a+r=r+ a=o$，记$r$为$-a$；
5. 乘法结合律：$\forall a,b,c\in R$，有$a\cdot(b\cdot c)=(a\cdot b)\cdot c$；
6. 乘法分配律：$\forall a,b,c\in R$，有$a\cdot(b+c)=a\cdot b+a\cdot c$、$(a+b)\cdot c=a\cdot c+b\cdot c$。

则称$R$关于运算$+,\cdot$构成一个*环*（Ring），记作$(R,+,\cdot)$。
