#热力学 

# 热力学第三定律
热力学第三定律是从低温现象研究中总结出的一个普遍的经验规律。
它有两种等价的表述方式：
- 绝对零度不可达原理：不可用有限次操作达到绝对零度；
- Nerst热定理：当温度趋于绝对零度时$$\lim_{ T \to 0 } S=S_{0}$$其中$S_{0}$是常数。
# 热力学第三定律的推论
## 膨胀系数
$$\lim_{ T \to 0 } \alpha=0$$
## 压强系数
$$\lim_{ T \to 0 } \beta=0$$
## 热容
$$\lim_{ T \to 0 } C_{V}=\lim_{ T \to 0 } C_{P}=0$$在[[晶体的热容]]中，Einstein、Debye都通过量子统计理论，验证了这个结果。
