#量子力学 
# 一维线性谐振子
先考虑一维的情况。
设质量为$m$的一维线性谐振子在运动时会受到$F=-Kx$的回复力，令$\omega=\sqrt{ \dfrac{K}{m} }$，则此时Hamiton量算符为$$\hat H =-\frac{\hbar^2}{2m}\frac{\mathrm{d}^2 }{\mathrm{d} x^2} +\frac{1}{2}m\omega^2x^2 $$它也是不含时的，故我们关键在于解定态Schrödinger方程$$\left( -\frac{\hbar^2}{2m}\frac{\mathrm{d}^2 }{\mathrm{d} x^2} +\frac{1}{2}m\omega^2x^2 \right)\psi(x)=E\psi(x)$$
## 坐标表象
可以通过一系列复杂的变换，得知它属于第一类Weber方程，最终得到：
- 本征函数：$$\psi_n(x)=N_ne^{-\frac{\alpha^2}{2}x^2 }H_n(\alpha x),N_n=\left ( \frac{\alpha}{\pi^{\frac{1}{2} } 2^n n!}  \right ) ^{\frac{1}{2} }$$其中$H_{n}$是Hermite函数。
- 能级：$$E_n=\hbar\omega\left(n+\frac{1}{2}\right),n=1,2,\cdots$$
## 占有数表象
不过除了上述基于级数的繁琐解法，我们还有另一种代数解法。
体系的Hamilton量为$$H=\frac{1}{2m}p^2+\frac{1}{2}m\omega^2x^2$$若做代换$$X=\sqrt{ \frac{m\omega}{\hbar} },\quad P=\frac{1}{\sqrt{ m\omega \hbar }}p$$则Hamilton量变为$$H=\frac{\hbar \omega}{2}(X^2+P^2)$$它具有$X-P$空间的旋转对称性。根据这一旋转的特征，引入两个算符：
- 湮灭算符$\hat{a}$：$$\hat{a}=\frac{1}{\sqrt{ 2 }}(\hat{X}+i\hat{P})$$
- 产生算符$\hat{a}^{\dagger}$：$$\hat{a}^{\dagger}=\frac{1}{\sqrt{ 2 }}(\hat{X}-i\hat{P})$$

根据$[\hat{X},\hat{P}]=i$，可以得到二者的对易关系$$[\hat{a},\hat{a}^{\dagger}]=1$$此外再定义粒子数算符$$\hat{N}=\hat{a}^{\dagger}\hat{a}$$则$$\hat{H}=h\omega\left( \hat{N}+\frac{1}{2} \right)$$而能量的本征值直接和粒子数$N$的本征值相关$$E=h\omega\left( N+\frac{1}{2} \right)$$至此我们将问题归结到求解粒子数本征态的问题。
粒子占有数算符有以下重要性质：
- **粒子占有数算符与Fourier变换算符对易**，即$$[\hat{N},\mathcal{F}]=0$$

---
设$\hat{N}$的归一化本征态为$\ket{n}$，满足$\hat{N}\ket{n}=n\ket{n}$。
1. 证明$n \geq 0$：考虑湮灭算符作用后的模方可发现$$|\hat{a}\ket{n} |^2=\bra{n} \hat{a}^{\dagger}\hat{a}\ket{n} =\bra{n} \hat{N}\ket{n} =n\geq 0$$
2. 粒子的产生和湮灭：
	计算对易子
	- $[\hat{N},\hat{a}]=[\hat{a}^{\dagger}\hat{a},\hat{a}]=\hat{a}^{\dagger}[\hat{a},\hat{a}]+[\hat{a}^{\dagger},\hat{a}]\hat{a}=-a\implies \hat{N}\hat{a}=\hat{a}(\hat{N}-1)$。
	- $[\hat{N},\hat{a}^{\dagger}]=\hat{a}^{\dagger}\implies \hat{N}\hat{a}^{\dagger}=\hat{a}^{\dagger}(\hat{N}+1)$。
将$\hat{N} \hat{a}$作用于$\ket{n}$，则$$\hat{N}(\hat{a}\ket{n} )=\hat{a}(\hat{N}-1)\ket{n} =\hat{a}(n-1)\ket{n} =(n-1)(\hat{a}\ket{n} )$$这说明：经过$\hat{a}$的作用以后，$\hat{a}\ket{n}$也是$\hat{N}$的本征态，只不过对应的本征值从$n$变成了$n-1$，就相当于是说少了一个粒子。
同样地，$\hat{a}^{\dagger}$作用后会把本征值从$n$变成$n+1$，就相当于说是多了一个粒子。
可以推导出$$\begin{equation}
\left\{
\begin{aligned}
&\hat{a}\ket{n} =\sqrt{ n }\ket{n-1}\\
&\hat{a}^{\dagger}\ket{n} =\sqrt{ n }\ket{n+1} 
\end{aligned}
\right.
\end{equation}$$
3. 确定基态：
湮灭算符$\hat{a}$每作用一次，本征值都会减少1，因为$n\geq 0$，这肯定不能无限进行下去，必然有一个基态，使得$$\hat{a}\ket{0} =0$$在左边作用一个$\hat{a}^{\dagger}$，可以发现$$\hat{a}^{\dagger}\hat{a}\ket{0} =\hat{N}\ket{0} =0$$也就是此时粒子数的本征值为0。
这就确定了基态能量$$E=h\omega\left( 0+\frac{1}{2} \right)=\frac{1}{2}\hbar \omega$$
4. 确立激发态：
不断运用产生算符$\hat{a}^{\dagger}$，则可以得到各级激发态，即$$\ket{n} =\frac{1}{\sqrt{ n! }}\ket{0} $$同时能得到各级能谱$$E_{n}=h\omega\left(n+ \frac{1}{2} \right)$$
# 三维线性谐振子
## 直角坐标系
- 本征函数：$$\psi_{n_{x},n_{y},n_{z}}(x,y,z)=\phi_{n_{x}}(x)\phi_{n_{y}}(y)\phi_{n_{z}}(z)$$各向为Hermite-Gauss函数$$\phi_{n}(x)=N_ne^{-x^2/2 }H_n( x),N_n=\left ( \frac{1}{\pi^{\frac{1}{2} } 2^n n!}  \right ) ^{1/2 }$$
- 本征值：$$E=\hbar \omega\left( n_{x}+n_{y}+n_{z}+\frac{3}{2} \right)$$
## 柱坐标系
- 本征函数：$$\psi_{n_{\rho},n_{\varphi},n_{z}}(\rho,\varphi,z)=R_{n_{\rho},n_{\varphi}}(\rho)\Phi_{n_{\varphi}}(\varphi)Z_{n_{z}}(z)$$径向为Laguerre-Gauss函数$$R_{n_{\rho},n_{\varphi}}(\rho)=N_{n_{\rho},n_{\varphi}}\rho^{|n_{\varphi}|}e^{ -\rho^2/2 }L_{n_{\rho}}^{|n_{\varphi}|}(\rho^2),N_{n_{\rho},n_{\varphi}}=\left( \frac{2n_{\rho}!}{(n_{\rho}+|n_{\varphi}|)!} \right)^{1/2}$$角向为简谐函数$$\Phi(\varphi)=\frac{1}{\sqrt{ 2\pi }}e^{ in_{\varphi}\varphi }$$柱向为Hermite-Gauss函数$$\phi_{n}(z)=N_ne^{-z^2/2 }H_n(z),N_n=\left ( \frac{1}{\pi^{\frac{1}{2} } 2^n n!}  \right ) ^{1/2 }$$
- 本征值：$$E=\hbar \omega\left[ (2n_{\rho}+|n_{\varphi}|+1)+\left( n_{z}+\frac{1}{2} \right) \right]$$
## 球坐标系
- 本征函数：$$\psi_{nlm}(r,\theta,\varphi)=R_{nlm}(r)Y_{lm}(\theta,\varphi)$$径向为广义Laguerre-Gauss函数（半整数阶）$$R_{nlm}=N_{nlm}r^{l}e^{ -r^2/2 }L_{n_{r}}^{l+1/2}(r^2),N_{nlm}=\left( \frac{2n!}{\Gamma(n+l+3/2)} \right)$$角向为球谐函数。
