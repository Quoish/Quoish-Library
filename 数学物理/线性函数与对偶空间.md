#线性代数 
# 线性函数
## 线性函数的定义
设$V$是域$F$上的线性空间，由于域$F$也可以看成一个1维的线性空间，因此自然可以考虑线性空间$V$到$F$的线性映射，我们把这种线性映射称为$V$上的线性函数。详细地说有：

---
设$V$是域$F$上线性空间，$V$到$F$的一个映射$f如$果满足：
1. $f(\alpha+\beta)=f(\alpha)+f(\beta)， \forall\alpha,\beta\in V$。
2. $f(k\alpha)=kf(\alpha)，\forall \alpha \in V,k\in F$。
则称$f$是$V$上的一个*线性函数*。

例如：求迹操作$\mathrm{tr}$是$M_{n}(F)$上的一个线性函数，称作迹函数。
再例如：对于$\mathbb{R}$上一个元素都是列矢量的线性空间$V$来说，行矢量（加上矩阵乘法）是一个线性函数，因为假如一个列矢量$\alpha=\begin{pmatrix}a \\ b\end{pmatrix} \in V$，行矢量$\alpha^*=\begin{pmatrix}c & d\end{pmatrix}$，它们相乘可以得到$$\alpha^* \alpha=ac+bd \in \mathbb{R}$$这就相当于$\alpha^*$把一个$V$中的元素映射到了数域$\mathbb{R}$。作为线性函数的$\alpha^*$也叫做$\alpha$的*对偶矢量*。
- 线性函数存在性：任给数域$F$中$n$个元素$a_1,a_2,…,a_n$，则存在唯一线性函数$f$使得$f(\alpha_i)=a_i,i=1,2,\cdots,n$。这个可以通过线性映射的存在性证明。
## 线性函数的表达式
设$\dim V = n$，$V$中任取一个基$α_1,α_2,…,α_n$，则$\forall \alpha\in V$，有$\alpha=x_1α_1+x_2α_2+…+x_nα_n$，设$f$是$V$上的一个线性函数，则称$$f(\alpha)=x_1f(α_1)+x_2f(α_2)+…+x_nf(α_n)$$为$f$在基$α_1,α_2,…,α_n$下的表达式。
# 线性函数空间和对偶空间
## 对偶空间
线性空间$V$上全体线性函数的集合$\mathrm{Hom}(V,F)$称作*线性函数空间*。
如果$V$是有限维的，设$\dim V=n$，则记$\mathrm{Hom}(V,F)$为$V^*$称$V^*$是$V$的*对偶空间*。
如果$V$是无穷维的，那么$V$的线性函数空间就不一定是$V$的对偶空间了，参见群表示论。
容易证明二者的同构关系$V^* \cong V$。
## 对偶空间的基
如果从$V$中取一个基$α_1,α_2,…,α_n$，如果要构造$V^*$的一个基，可以设：
$$
\begin{equation}
\left.
\begin{aligned}
&f_1使得f_1(α_1)=1,f(α_i)=0,i\neq1\\
&f_2使得f_2(α_2)=1,f(α_i)=0,i\neq2\\
&\vdots\\
&f_n使得f_n(α_n)=1,f(α_i)=0,i\neq n\\
\end{aligned}
\right.
\end{equation}$$
可以从$k_1f_1+k_2f_2+\cdots+k_nf_n\equiv0$推出系数全零，证明这些线性函数线性无关，由于$\dim V^*=n$，故$f_1,f_2,\cdots,f_n$是$V^*$的一个基，称$f_1,f_2,\cdots,f_n$为$V$的基$α_1,α_2,…,α_n$的*对偶基*。
- *基与对偶基的正交性*：$$f_i(\alpha_{j})=\delta_{ij},i,j\in\{1,2,\cdots,n\}$$

我们可以对比线性空间和它的对偶空间中的元素被基底表示的样子：
- $V$中任一向量$\alpha=\sum_{i=1}^nx_i\alpha_i$，则$$f_j(\alpha)=\sum_{i=1}^nx_if_j(\alpha_i)=x_j$$因此$V$中任意向量$α$可写作$$\alpha=\sum_{i=1}^nf_i(\alpha)\alpha_i$$其中$f_{i}$是$\alpha_{i}$的对偶基。
- $V^*$中任一函数$f=\sum_{i=1}^nk_if_i$，则$$f(\alpha_j)=\sum_{i=1}^nk_if_i(\alpha_j)=k_j$$因此$V^*$中任意函数$f$可写作$$f=\sum_{i=1}^nf(\alpha_i)f_i$$其中$\alpha_{i}$是$f_{i}$的对偶基。

可见，这就是[[Fourier变换]]与反变换在更一般意义上的形式。
我们姑且直接用上面的结论讨论无穷维的形式，于是可以发现$$\alpha=\sum_{i=1}^nf_i(\alpha)\alpha_i\implies f(t)=\int F(\omega)e^{ i\omega t } \frac{\mathrm{d}\omega}{2\pi}$$$$f=\sum_{i=1}^nf(\alpha_i)f_i\implies F(\omega)=\int f(t)e^{ -i\omega t }\mathrm{d}t$$$$f_{i}(\alpha_{j})=\delta_{ij}\implies \langle e^{ i\omega t },e^{ -i\omega' t} \rangle=2\pi\delta(\omega-\omega') $$
也就是说，**时域函数空间与频域函数空间构成一对对偶空间**。
## 对偶基的变换
对偶空间中有与线性空间类似的基变换，并且二者的变换矩阵存在着关联。
线性空间的基变换：$n$维线性空间$V$的两个基$α_1,α_2,…,α_n$和$β_1,β_2,…,β_n$，有过渡矩阵$A$使得$$(β_1,β_2,…,β_n)=(α_1,α_2,…,α_n)A$$对偶空间的基变换：$V$的对偶空间$V^*$的两个基分别是$f_1,f_2,…,f_n$和$g_1,g_2,…,g_n$，有过渡矩阵$B$使得$$(g_1,g_2,…,g_n)=(f_1,f_2,…,f_n)B$$
如果$α_1,α_2,…,α_n$的对偶基恰是$f_1,f_2,…,f_n$，$β_1,β_2,…,β_n$的对偶基恰是$g_1,g_2,…,g_n$，那么$$\boxed{B=(A^{-1})^T}$$如果是复矩阵则变为$$\boxed{B=(A^{-1})^{\dagger}}$$
从这个结论的视角我们可以重新审视Fourier变换的尺变性质（$\mathscr{F}[f(at)]=\dfrac{1}{|a|}F(\dfrac{j\omega}{a})$），频域自变量中除以了$a$本质上是时域自变量乘以$a$的逆，而前面的$1/|a|$保持了$L^2$测度下内积的不变。
# 双重对偶空间
因为$V^*$也跟$V$一样，也是$n$维的线性空间，所以也可以考虑$V^*$的对偶空间。把$V^*$的对偶空间$(V^*)^*$记作$V^{**}$，称$V^{**}$为$V$的*双重对偶空间*。
很显然的是三个空间都是相互同构的$\dim V=\dim V^*=\dim V^{**}=n$，
$V$中取一个基$α_1,α_2,…,α_n$，$V^*$中取一个基$f_1,f_2,…,f_n$，$V^{**}$中取一个基$α^{**}_1,α^{**}_2,…,α^{**}_n$，我们可以建立以下这样的关系：$$\begin{matrix}
\begin{align}
&V\xrightarrow{\ \ \ \ \ \ \ \ \ \ 同构映射\sigma\ \ \ \ \ \ \ \ }V^*\xrightarrow{\ \ \ \ \ \ \ \ \ \ 同构映射\tau\ \ \ \ \ \ \ \ }V^{**}\\
\alpha=&\sum_{i=1}^{n}x_i\alpha_i \xmapsto{\ \ \ \ \ \ \ \ \ }f=\sum_{i=1}^{n}x_if_i \xmapsto{\ \ \ \ \ \ \ \ \ } \sum_{i=1}^{n}x_i\alpha_i^{**}:=\alpha^{**} \\
\end{align}
\end{matrix}$$通过这样双重的同构映射，我们就完成了对$V^{**}$中元素的刻画。
但$V^{**}$中的元素到底是什么呢？我们可以看任给$f\in V^*$有：$$\begin{matrix}
\begin{align}
\alpha^{**}(f)&=\sum_{i=1}^nx_i\alpha^{**}_i(f)\\
&=\sum_{i=1}^nx_i\alpha^{**}_i(\sum_{i=1}^nf(\alpha_i)f_i)\\
&=\sum_{i=1}^nx_if(\alpha_i)=f(\alpha)\in F
\end{align}
\end{matrix}$$所以我们可以看出来：$V^{**}$**中的元素是把函数映到数域的函数，是函数的函数，也就是泛函**。
上面可以看到，存在一个从$V$到$V^{**}$的双射$\sigma \circ\tau$把$α$映到$α^{**}$，其中$α^{**}(f)=f(α)$。我们不需要依赖具体的基，依然可以实现$V$和$V^{**}$中元素的一一对应，这种不依赖于基的选择的同构称作*自然同构*。
因此可以把$α$和$α^{**}$**等同起来**，可以把$V$和$V^{**}$**等同起来**，视作同一回事，于是$V$可以看作$V^*$的对偶空间，进而$V$和$V^*$**互为对偶空间**。
这种把矢量$\alpha$和泛函$\alpha^{**}$等量齐观的观点，直接成为我们推广[[流形上的矢量]]的出发点。所以到现在我们可以说，**矢量就是泛函，泛函就是矢量**。
