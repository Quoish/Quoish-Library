# 概率论基础概念
## 随机试验
*随机试验*指在相同条件下对某随机现象进行的大量重复观测。
其特点有：
1. 可以在相同的条件下重复的进行。
2. 试验的结果可能不止一个，并且能事先预知所有可能的结果。
3. 进行试验之前不知道结果会是什么。
## 样本空间
- *样本空间*：随机事件$E$所有可能结果组成的集合，记为$S$。
- *样本点*：样本空间的元素。
## 随机事件
- *随机事件*：试验$E$的样本空间$S$中的子集为$E$的随机事件。*事件发生*则为当子集中一个样本点出现时。
- *基本事件*：一个样本点组成的单点集
- *必然事件*：样本空间$S$。
- *不可能事件*：和样本空间的交集为$\varnothing$的事件。
# 事件关系
事件关系有以下几种类型：
1. 包含关系：$A\subseteq B$。
2. 和事件：$A\cup B$、$A+B$。
3. 积事件：$A\cap B$、$AB$。
4. 差事件：$A-B$。
5. 互不相容性、互斥性：$A\cap B=\varnothing$。
6. 逆事件、对立事件：$\bar{A}$。
