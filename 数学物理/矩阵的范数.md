# 诱导范数
诱导范数将矩阵视为线性变换，度量它能把向量“拉伸”的最大倍数。
$m\times n$的矩阵$A$的诱导$p-$范数定义为$$||A||_{p}=\sup_{x\neq 0} \frac{||Ax||_{p}}{||x||_{p}}$$
常见的形式有以下几种：
## 1-范数（列和范数）
$$||A||_{1}=\max_{1\leq j \leq n}\sum_{i=1}^m |a_{ij}|$$即矩阵所有列向量的绝对值之和的最大值，反映了矩阵对向量的最大“列”作用。
## 2-范数（谱范数）
$$||A||_{2}=\sigma_{\max}(A)$$
即$A$的最大奇异值，几何上表示矩阵作为线性变换的最大拉伸因子，常用于分析矩阵的稳定性和条件数。
## ∞-范数（行和范数）
$$||A||_{\infty}=\max_{1\leq i\leq m}\sum_{1\leq j \leq \infty}^n|a_{ij}|$$
