#电磁学 
# 一般情况的边界条件

|   场   |                           边界条件                           |           边界条件           |
| :---: | :------------------------------------------------------: | :----------------------: |
|  磁场   | $\vec{e}_{n}\times(\vec{H}_{1}-\vec{H}_{2})=\vec{J}_{S}$ |  $H_{1t}-H_{2t}=J_{s}$   |
|  电场   |      $\vec{e}_{n}\times(\vec{E}_{1}-\vec{E}_{2})=0$      |    $E_{1t}-E_{2t}=0$     |
| 磁感应强度 |      $\vec{e}_{n}\cdot(\vec{B}_{1}-\vec{B}_{2})=0$       |    $B_{1n}-B_{2n}=0$     |
| 电位移矢量 |   $\vec{e}_{n}\cdot(\vec{D}_{1}-\vec{D}_{2})=\rho_{S}$   | $D_{1n}-D_{2n}=\rho_{S}$ |
其中下标为1的是矢量$\vec{e}_{n}$箭头尖端处的场，下标为2的是矢量$\vec{e}_{n}$箭头根端处的场。
此边界条件无论对于静态电磁场还是时变电磁场都普遍成立。
## 磁场边界条件
![[电磁场界面环路.png]]
取小矩形$abcd$，做[[Maxwell方程组#全电流Ampere环路定理]]
$$\oint_{C}\vec{H}\cdot \mathrm{d}\vec{l}=\iint_{\Delta S}\vec{J}\cdot \mathrm{d}\vec{S}+\iint_{\Delta S}\frac{ \partial  \vec{D} }{ \partial  t}\cdot \mathrm{d}\vec{S} $$
因为界面附近的$\vec{H}$必定是有限值，令$\Delta h=bc=da$，当$\Delta h\to 0$的时候，$da$，$bc$的积分必定是0，于是有
$$\lim_{ \Delta h \to 0 } \oint_{C}\vec{H}\cdot \mathrm{d}\vec{l}=\int_{ab}\vec{H}\cdot \mathrm{d}\vec{l}+\int_{cd}\vec{H}\cdot \mathrm{d}\vec{l}$$
$\vec{D}$是连续变化的，$\frac{ \partial  \vec{D} }{ \partial  t}$也必定是有限值，当$\Delta h\to 0$的时候，该曲面积分也是0。如果存在面电流的话，则
$$\lim_{ \Delta h \to 0 } \iint_{\Delta S}\vec{J}\cdot \mathrm{d}\vec{S}=\int_{\Delta l}(\lim_{ \Delta h \to 0 } \Delta h \vec{J})\cdot \vec{e}_{p}\mathrm{d}\vec{l}=\int_{\Delta l}\vec{J}_{S}\cdot \vec{e}_{p}\mathrm{d}\vec{l}$$
因此
$$\int_{\Delta l}(\vec{H}_{1}-\vec{H}_{2})\cdot \vec{e}_{t}\mathrm{d}l=\int_{\Delta l}\vec{J}_{S}\cdot \vec{e}_{p}\mathrm{d}l$$
当$\Delta l$很小时，可以认为
$$(\vec{H}_{1}-\vec{H}_{2})\cdot \vec{e}_{t}=\vec{J}_{S}\cdot \vec{e}_{p}$$
借$\vec{e}_{t}=\vec{e}_{p}\times \vec{e}_{n}$，化为
$$\vec{e}_{n}\times(\vec{H}_{1}-\vec{H}_{2})=\vec{J}_{S}$$
由此可见，当分界面上存在自由面电流时，分界面上两侧的磁场强度矢量的切向分量是不连续的。
若两种媒质的电导率均为有限值，则媒质内的传导电流密度$\vec{J}=σ\vec{E}$也为有限值，于是$\vec{J}_{S}=\lim_{ \Delta h \to 0 }\vec{J}\Delta h=0$，即电导率为有限值的两种媒质分界面上不存在面电流，因此有
$$\vec{e}_{n}\times(\vec{H}_{1}-\vec{H}_{2})=0$$
## 电场边界条件
同样用上面那个小矩形，运用[[Maxwell方程组#Faraday电磁感应定理]]。
用类似的方法讨论$\vec{E}$，$\vec{B}$，$\frac{ \partial  \vec{B} }{ \partial  t}$作为有限值$\Delta h\to 0$的状况，可以得到
$$\vec{e}_{t}\cdot(\vec{E}_{1}-\vec{E}_{2})=0$$
也可以写成叉乘
$$\vec{e}_{n}\times(\vec{E}_{1}-\vec{E}_{2})=0$$
由此可见，无论分界面上是否存在面分布电荷，两侧的电场强度矢量E的切向分量总是连续的。
## 磁感应强度边界条件
![[电磁场界面圆柱.png]]
取小圆柱，做[[Maxwell方程组#磁场的Gauss定理]]。
因为分界面的$\vec{B}$为有限值，所以
$$\lim_{ \Delta h \to 0 }\iint_{侧面}\vec{B}\cdot \mathrm{d}\vec{S}=0$$
进而$$\lim_{ \Delta h \to 0 } \iint_{S}\kern{-17.8pt}\subset\kern{-2pt}\supset    \vec{B}\cdot \mathrm{d}\vec{S}=\iint_{上底}\vec{B}_{1}\cdot \mathrm{d}\vec{S}+\iint_{下底}\vec{B}_{2}\cdot \mathrm{d}\vec{S}=\iint_{\Delta S}(\vec{B}_{1}-\vec{B}_{2})\cdot \vec{e}_{n}\mathrm{d}S=0$$
于是
$$\vec{e}_{n}\cdot(\vec{B}_{1}-\vec{B}_{2})=0$$
由此可见，无论分界面上是否存在面分布电流，磁感应强度的法向分量在分界面上总是连续的。
## 电位移矢量边界条件
同样用上面那个小圆柱，运用[[Maxwell方程组#电场的Gauss定理]]。
类似有$$\lim_{ \Delta h \to 0 }\iint_{侧面}\vec{D}\cdot \mathrm{d}\vec{S}=0$$
推出$$\lim_{ \Delta h \to 0 } \iint_{S}\kern{-17.8pt}\subset\kern{-2pt}\supset    \vec{D}\cdot \mathrm{d}\vec{S}=\iint_{\Delta S}(\vec{D}_{1}-\vec{D}_{2})\cdot \vec{e}_{n}\mathrm{d}S= \iint_{\Delta S}\lim_{ \Delta h \to 0 }(\Delta h\rho) \mathrm{d}S=\iint_{\Delta S}\rho_{S}\mathrm{d}S$$
故
$$\vec{e}_{n}\cdot(\vec{D}_{1}-\vec{D}_{2})=\rho_{S}$$
由此可见，分界面上存在自由电荷面密度时，电位移矢量的法向分量在分界面上是不连续的。
# 理想导体的边界条件
理想导体指电导率非常大的导体，即认为$\sigma=\infty$。
为了避免出现无限大的电流，所以电场必须恒为0。
因为电场为0，所以磁场必须恒定。
总结理想导体必须：
- $\sigma=\infty$
- $\vec{E}=0,\quad \vec{D}=0$
- $\dfrac{ \partial  \vec{B} }{ \partial  t}=0,\quad \dfrac{ \partial  \vec{H} }{ \partial  t}=0$

设介质2是理想导体，内部磁场恒定，不妨设为0，于是边界条件有

|   场   |                    边界条件                     |
| :---: | :-----------------------------------------: |
|  磁场   | $\vec{e}_{n}\times \vec{H}_{1}=\vec{J}_{S}$ |
|  电场   |      $\vec{e}_{n}\times \vec{E}_{1}=0$      |
| 磁感应强度 |      $\vec{e}_{n}\cdot \vec{B}_{1}=0$       |
| 电位移矢量 |   $\vec{e}_{n}\cdot \vec{D}_{1}=\rho_{S}$   |
# 理想介质的边界条件
理想介质指电导率非常小的导体，即认为$\sigma=0$
由于理想介质不存在自由电子，所以$\rho_{S}=0,\vec{J}_{S}=0$
于是边界条件有

|   场   |                      边界条件                      |
| :---: | :--------------------------------------------: |
|  磁场   | $\vec{e}_{n}\times(\vec{H}_{1}-\vec{H}_{2})=0$ |
|  电场   | $\vec{e}_{n}\times(\vec{E}_{1}-\vec{E}_{2})=0$ |
| 磁感应强度 | $\vec{e}_{n}\cdot(\vec{B}_{1}-\vec{B}_{2})=0$  |
| 电位移矢量 | $\vec{e}_{n}\cdot(\vec{D}_{1}-\vec{D}_{2})=0$  |
