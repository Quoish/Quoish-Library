#随机过程 
# 随机过程的极限
## 随机变量序列的极限
随机变量不同于普通变量，它有许多种极限的定义，最常见的是以下两种。
### 依概率收敛
> 随机变量序列$\{ X_{n} \}_{n=1}^{\infty}$，$\forall \varepsilon>0$，都有$$\lim_{ n \to \infty } P\{ |X_{n}-X| \geq\varepsilon\}=0$$则称$\{ X_{n} \}$依概率收敛于随机变量$X$，记作$$\lim_{ n \to \infty } X_{n} \overset{P}{=}X$$
### 均方收敛
设随机变量$X$和随机变量序列$\{ X_{n} \}_{n=1}^{\infty}$有二阶矩，若有$$\lim_{ n \to \infty } E[|X_{n}-X|^2]=0$$则称随机变量序列$\{ X_{n} \}_{n=1}^{\infty}$均方收敛于$X$，记作$$\mathrm{l.i.m\ }X_{n}=X$$其中l.i.m.是limit in mean square的缩写。
由Chebyshev不等式$$P\{ |X_{n}-X|\geq \varepsilon \}\leq \frac{E[|X_{n}-X|^2]}{\varepsilon^2}$$可知，**均方收敛一定依概率收敛**。
## 随机过程的极限
### 依概率收敛
随机过程$X(t)$在$t\to t_{0}$时，$\forall \varepsilon>0$恒有$$\lim_{ t \to t_{0} } P\{ |X(t)-X|\geq \varepsilon\}=0$$则称随机过程$X(t)$在$t\to t_{0}$时依概率收敛于随机变量$X$，记作$$\lim_{ t \to t_{0} } X(t) \overset{P}{=}X$$
### 均方收敛
设随机变量$X$和随机过程$X(t)$有二阶矩，若有$$\lim_{ n \to \infty } E[|X_{n}-X|^2]=0$$则称随机变量序列$\{ X_{n} \}_{n=1}^{\infty}$均方收敛于$X$，记作$$\mathrm{l.i.m\ }X_{n}=X$$
# 随机过程的连续性
## 均方连续的定义
若随机过程$X(t)$在$t$点附近有$$\lim_{ \Delta t \to 0 } E[|X(t+\Delta t)-X(t)|^2]=0$$或$$\mathrm{l.i.m.}_{\Delta t\to 0}X(t+\Delta t)=X(t)$$则称$X(t)$在$t$*均方连续*。
## 均方连续的条件
实随机过程$X(t)$在$t$点均方连续的充要条件是：
- 其相关函数$R_{X}(t_{1},t_{2})$在$t_{1}=t_{2}=t$时连续。

若是平稳过程，则充要条件为相关函数在$\tau=0$处连续，即$$\lim_{ \tau \to 0 } R_{X}(\tau)=R_{X}(0)$$
## 均值连续
若随机过程$X(t)$均方连续，则其均值$E[X(t)]$必定连续，即$$\lim_{ \Delta t \to 0 } E[X(t+\Delta t)]=E[X(t)]$$
均方连续时，极限和均值可以换序，即$$\lim_{ \Delta t \to 0 } E[X(t+\Delta t)]=E[\mathrm{l.i.m._{\Delta t \to 0}}X(t+\Delta t)]$$
# 随机过程的微分
## 均方导数的定义
设有随机过程$X(t)$，若当$\Delta t\to0$时，增量比值$[X(t+\Delta t)-X(t)]/\Delta t$依均方收敛于某随机变量（与$t$有关），则此随机变量称为随机过程$X(t)$在$t$点的均方导数，记作$\mathrm{d}X(t)/\mathrm{d}t$，或简记为$\dot{X}(t)$，即$$\dot{X}(t)=\frac{ \mathrm{d}  X(t) }{ \mathrm{d}  t} =\mathrm{l.i.m.}_{\Delta t \to 0} \frac{X(t+\Delta t)-X(t)}{\Delta t}$$或者$$\lim_{ \Delta \to 0 } E\left[\left| \frac{X(t+\Delta t)-X(t)}{\Delta t}-\dot{X}(t) \right|^2   \right]=0$$
## 均方可微的条件
随机过程$X(t)$均方可微的充要条件为：相关函数$R_{X}(t_{1},t_{2})$在自变量$t_{1}=t_{2}=t$时，存在二阶混合偏导数$\dfrac{ \partial^2 R_{X}(t_{1},t_{2})  }{ \partial t_{1}\partial t_{2} }$。
若是平稳过程，则充要条件为：在$\tau=0$时，相关函数$R_{X}(\tau)$的二阶导数$\left. \dfrac{ \mathrm{d}  ^2 R_{X}(\tau) }{ \mathrm{d}  \tau^2} \right|_{\tau=0}$存在。
## 随机过程的微分变换
### 随机过程均方导数的均值
随机过程导数的均值等于均值的导数，$$E\left[ \frac{ \mathrm{d}  X(t) }{ \mathrm{d}  t}  \right]=\frac{ \mathrm{d}   }{ \mathrm{d}  t} E[X(t)]$$可见随机过程的导数和均值运算可以换序。
若$X(t)$是平稳过程，则$\mu_{X}(t)=c$是常数，所以$\dfrac{ \mathrm{d}  X(t) }{ \mathrm{d}  t}$的均值是0。
### 随机过程均方导数的相关函数
$$R_{\dot{X}}(t_{1},t_{2})=\frac{ \partial   }{ \partial  t_{1}} R_{X\dot{X}}(t_{1},t_{2}),\quad R_{\dot{X}}(t_{1},t_{2})=\frac{ \partial   }{ \partial  t_{2}} R_{\dot{X}X}(t_{1},t_{2})$$可见$$R_{\dot{X}}(t_{1},t_{2})=\frac{ \partial  ^2R_{X}(t_{1},t_{2}) }{ \partial  t_{1}\partial t_{2}} $$
若$X(t)$是平稳过程，且$\tau=t_{2}-t_{1}$则$$R_{\dot{X}}(\tau)=-\frac{ \mathrm{d}^2  R_{X}(\tau) }{ \mathrm{d}  \tau^2} $$
### 随机过程均方导数的相关函数
$X(t)$和$\dot{X}(t)$的功率谱密度函数的关系为$$G_{\dot{X}}(\omega)=\omega^2G_{X}(\omega)$$
# 随机过程的积分
## 均方积分的定义
随机过程$X(t)$的任一样本$x_{i}(t)$是时间$t$的确定函数，今在确定区间$(a,b)$上对它积分，积分值$y$是其所有子区间上面积之和的极限，即$$y=\int_{a}^{b}x_{i}(t)\mathrm{d}t=$$式中：$\Delta t_{k}=t_{k}-t_{k-1}$。
显然，对于任一样本$x_{i}(t)$，其积分值y都是定值，但样本不同则其积分值$y$也就不同，因而随机过程在确定区间上的积分值是一个随机变量。在依均方收敛意义下，过程的积分值才是定值，可见随机过程$X(t)$的均方积分可作如下定义：
若有$$\lim_{ \Delta t_{k} \to 0 } \left[ \left| \sum_{k=1}^{n}X(t_{k})\Delta t_{k}-\int_{a}^{b}X(t)\mathrm{d}t  \right|^2 \right]=0 $$则称随机变量$$Y=\int_{a}^{b}X(t)\mathrm{d}t=\mathrm{l.i.m.}_{\Delta t_{k}\to 0}\sum_{k=1}^{n}X(t_{k})\Delta t_{k}$$为随机过程$X(t)$在确定区间$(a,b)$上的均方积分。
应该指出，虽然此处仍用一般确知函数的积分符号，未加区分，但其含义已不同，在此它表示依均方收敛意义下的积分。
类似地同样可以推广出广义积分的定义。
## 随机过程的积分变换
### 随机过程均方导数的均值
随机过程积分的均值等于均值的积分，$$E\left[ \int_{0}^{t}X(a)\mathrm{d}a  \right]=\int_{0}^{t} E[X(a)]$$可见随机过程的积分和均值运算可以换序。
若$X(t)$是平稳过程，则$\mu_{X}(t)=c$是常数，所以$\int_{0}^tX(a)\mathrm{d}a$的均值是$ct$。可见，平稳过程的积分不再是平稳过程。
### 随机过程均方导数的相关函数
设$Y(t)=\int_{0}^{t}X(a)\mathrm{d}a$$$R_{XY}(t_{1},t_{2})=\int_{0}^{t_{2}} R_{X}(t_{1},a_{2})\mathrm{d}a_{2},\quad R_{YX}(t_{1},t_{2})=\int_{0}^{t_{1}} R_{X}(a_{1},t_{2})\mathrm{d}a$$$$R_{Y}(t_{1},t_{2})=\int_{0}^{t_{1}}\int_{0}^{t_{2}}R_{X}(a_{1},a_{2})\mathrm{d}a_{1}\mathrm{d}a_{2}$$
若$X(t)$是平稳过程，这时虽然相关函数$R_{X}(\tau)$，与时间$t$无关，但是过程$Y(t)$的自相关函数和互相关函数却都已与时间$t$有关。
